# 交互设计文档

**项目**：抖音关注博主筛选助手
**版本**：v1.0
**日期**：2024年

---

## 1. 设计原则

### 1.1 核心原则

| 原则 | 描述 | 实践 |
|------|------|------|
| 高效 | 操作步骤最少化 | 列表页支持快捷操作，一键筛选 |
| 清晰 | 信息层级分明 | 卡片突出关键信息，详细数据在详情页 |
| 可控 | 操作可撤销 | 所有操作有记录，支持撤销 |
| 即时 | 操作即时反馈 | 打标签/分层即时保存并显示 |

### 1.2 交互模式

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              交互模式                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  点击 (Tap)          单指点击                           选择/确认             │
│  长按 (Long Press)   按住 500ms                         显示快捷菜单         │
│  滑动手势            水平/垂直滑动                       快捷操作/筛选        │
│  下拉刷新            Pull to Refresh                     刷新列表             │
│  拖拽排序            Drag & Drop                         排序/重组            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 4. 动画规范

### 4.1 页面转场

| 页面类型 | 动画类型 | 时长 | 缓动函数 |
|---------|---------|------|----------|
| 推入/弹出 | 水平滑动 | 300ms | ease-out |
| 淡入/淡出 | 透明度变化 | 200ms | ease-in-out |
| 模态框 | 缩放+淡入 | 250ms | ease-out |
| 抽屉 | 水平滑动 | 300ms | ease-out |

### 4.2 元素动画

| 元素 | 动画类型 | 触发条件 | 时长 |
|------|---------|---------|------|
| 卡片选择 | 边框高亮 | 选中 | 150ms |
| 标签添加 | 弹入 | 添加成功 | 200ms |
| 标签移除 | 淡出+收缩 | 移除成功 | 150ms |
| 优先级变更 | 颜色渐变 | 变更成功 | 200ms |
| 列表项删除 | 左滑淡出 | 删除成功 | 200ms |
| 加载状态 | 骨架屏闪烁 | 加载中 | 1000ms循环 |

## 5. 异常处理

### 5.1 空状态

| 场景 | 界面 | 文案 |
|------|------|------|
| 无创作者 | 插画 + 按钮 | "还没有添加创作者" + "立即添加" |
| 筛选无结果 | 插画 | "没有符合条件的创作者" + "调整筛选条件" |
| 无今日推荐 | 插画 | "今日暂无推荐更新" |
| 无待处理任务 | 插画 | "太棒了！没有待处理任务" |

### 5.2 加载状态

| 场景 | 界面 |
|------|------|
| 列表加载 | 骨架屏（3个卡片占位） |
| 详情加载 | 骨架屏（头像+信息占位） |
| 操作处理中 | 按钮loading态 + 文字"处理中..." |

### 5.3 错误状态

| 错误类型 | 界面 | 操作 |
|---------|------|------|
| 网络错误 | Toast + 重试按钮 | 点击重试 |
| 保存失败 | Toast | 自动保存草稿 |
| 账号检测失败 | 状态显示"检测失败" | 提供手动刷新 |
