### 2.2 功能模块说明

#### 2.2.1 今日推荐模块

| 属性 | 说明 |
|------|------|
| 功能描述 | 展示S/A级博主中有今日更新的创作者 |
| 需求来源 | 用户需要快速找到当天值得看的内容 |
| 数据来源 | 从创作者列表筛选：priority in (S,A) AND last_updated = today |
| 展示逻辑 | 按更新时间倒序，最多展示20个 |
| 交互要求 | 点击卡片进入创作者详情页 |

**原型示意**：
```
┌─────────────────────────────────────────┐
│  今日推荐                      2024.01.15 │
├─────────────────────────────────────────┤
│                                         │
│  ┌────┐  ┌────┐  ┌────┐  ┌────┐      │
│  │头像│  │头像│  │头像│  │头像│      │
│  │ S  │  │ S  │  │ A  │  │ A  │      │
│  └────┘  └────┘  └────┘  └────┘      │
│  老高小术  所长林超  硬核何  科技美学  │
│  今天更新  今天更新  今天更新  今天更新  │
│                                         │
└─────────────────────────────────────────┘
```

**验收标准**：
- [ ] 正确展示S/A级今日更新博主
- [ ] 显示优先级标签
- [ ] 显示更新时间
- [ ] 点击可跳转详情页

## 5. 筛选面板原型需求说明

| 项目 | 内容 |

### 5.2 筛选维度说明

| 筛选维度 | 类型 | 选项 | 说明 |

| L1分类 | 多选 | 7个分类 | 知识/资讯/技能/审美/娱乐/情绪/其他 |

| 账号状态 | 多选 | 4种 | 正常/已注销/不可访问/异常 |
| 广告占比 | 范围 | 1-5 | 拖动滑块选择区间 |
| 风险标记 | 多选 | 4种 | 标题党/低质/强带货/情绪消耗 |

```

│  [ ]审美/颜值 [ ]娱乐/抽象 [ ]情绪    │
│                                         │
│  优先级                                │
│  [✓S级12] [✓A级45] [ ]B级120 [ ]C级 │
│                                         │
│  更新频率                              │
│  [ ]日更  [✓]周更  [ ]月更  [ ]停更  │
│                                         │
│  广告占比                              │
│  低 ●━━━━━━━━━━● 高                   │
│       1        2        3        4    │
│       当前范围: 1 - 3                  │
│                                         │
└─────────────────────────────────────────┘
```

**验收标准**：
- [ ] 正确展示所有筛选维度
- [ ] 每个维度可正确选择
- [ ] 实时显示符合条件的人数
- [ ] 重置按钮清空所有条件
- [ ] 应用按钮关闭面板并应用筛选

### 9.1 设置中心概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 设置中心 |
| 页面路径 | `/settings` |

### 9.2 功能模块说明

#### 9.2.1 账号与数据

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 导出数据 | 操作 | 导出创作者数据为CSV |
| 导入数据 | 操作 | 从CSV导入创作者 |
| 清除数据 | 操作 | 清除所有本地数据（需确认） |

#### 9.2.2 自动化设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 自动检测账号状态 | 开关 | 开启 | 定时检测创作者账号状态 |
| 自动取关已注销 | 开关 | 关闭 | 需用户确认开启 |
| 每周复盘提醒 | 开关 | 开启 | 每周日推送复盘提醒 |

#### 9.2.3 通知设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 推送通知 | 开关 | 开启 | App内通知 |
| 邮件通知 | 开关 | 关闭 | 发送至绑定邮箱 |
