## 2. 页面层级

### 2.1 一级导航

| 页面 | 路径 | 描述 | 优先级 |
|------|------|------|--------|

## 3. 核心数据模型

### 3.1 创作者（Creator）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              Creator                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│  基本信息                        │  分类信息                                │
│  ├── creator_id (PK)            │  ├── primary_tag (L1主类目)              │
│  ├── name (昵称)                │  ├── secondary_tags (L2赛道)            │
│  ├── platform (抖音)             │  ├── content_type (内容类型)             │
│  ├── profile_url (主页链接)      │  └── purpose (关注目的)                  │
│  ├── avatar (头像)               ├───────────────────────────────────────  │
│  └── bio (简介)                  │  评分信息                                 │
│                                  │  ├── info_density (信息密度 1-5)          │
│                                  │  ├── credibility (可信度 1-5)            │
│                                  │  ├── originality (原创度 1-5)            │
│                                  │  ├── actionability (可操作性 1-5)        │
│                                  │  ├── redundancy (重复度 1-5)             │
│                                  │  └── ad_ratio (广告占比 1-5)             │
│                                  ├───────────────────────────────────────  │
│                                  │  状态信息                                 │
│                                  │  ├── priority (S/A/B/C/D)               │
│                                  │  ├── status (正常/已注销/不可访问)        │
│                                  │  ├── update_frequency (更新频率)          │
│                                  │  └── last_updated (最后更新时间)         │
│                                  ├───────────────────────────────────────  │
│                                  │  用户数据                                 │
│                                  │  ├── notes (备注)                        │
│                                  │  ├── tags (用户自定义标签)                │
│                                  │  └── created_at / updated_at             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.2 规则（Rule）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                Rule                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│  ├── rule_id (PK)                                                         │
│  ├── name (规则名称)                                                       │
│  ├── description (规则描述)                                                 │
│  ├── condition (条件表达式)                                                  │
│  ├── action (操作类型: upgrade/downgrade/tag/review)                        │
│  ├── action_value (操作值)                                                 │
│  ├── priority (规则优先级)                                                  │
│  ├── enabled (是否启用)                                                     │
│  └── created_at / updated_at                                               │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.3 复核记录（ReviewLog）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                             ReviewLog                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│  ├── log_id (PK)                                                          │
│  ├── creator_id (FK)                                                      │
│  ├── date (操作日期)                                                       │
│  ├── user_action (操作类型)                                                 │
│  ├── reason (操作原因)                                                     │
│  ├── before_state (操作前状态快照)                                           │
│  └── after_state (操作后状态快照)                                           │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.4 标签（Tag）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                Tag                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│  ├── tag_id (PK)                                                          │
│  ├── name (标签名称)                                                       │
│  ├── type (L1/L2/用户自定义)                                               │
│  ├── parent_id (父标签ID，L2用)                                             │
│  ├── synonyms (同义词列表)                                                 │
│  ├── color (标签颜色)                                                      │
│  ├── usage_count (使用次数)                                                │
│  └── created_at / updated_at                                               │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 7. 搜索与筛选

### 7.1 搜索

| 搜索类型 | 范围 | 提示词 |
|---------|------|--------|
| 模糊搜索 | 昵称、备注 | "搜索创作者昵称或备注" |
| 精确搜索 | 主页链接 | 支持粘贴链接搜索 |

### 7.2 筛选维度

| 维度 | 类型 | 选项 |
|------|------|------|
| L1分类 | 多选 | 知识/资讯/技能/审美/娱乐/情绪/其他 |
| L2赛道 | 多选 | 抖音赛道标签 |
| 优先级 | 多选 | S/A/B/C/D |
| 更新频率 | 多选 | 日更/周更/月更/停更 |
| 账号状态 | 多选 | 正常/已注销/不可访问 |
| 广告占比 | 区间 | 1-5 |
| 风险标记 | 多选 | 标题党/低质/强带货/情绪消耗 |

## 9. 状态定义

### 9.1 创作者状态

| 状态 | 描述 | 显示 |
|------|------|------|
| 正常 | 可正常访问 | 🟢 |
| 已注销 | 账号已注销 | 🔴 |
| 不可访问 | 主页无法访问 | 🟡 |
| 疑似异常 | 需要复核 | 🟠 |

### 9.2 优先级定义

| 优先级 | 描述 | 颜色 |
|--------|------|------|
| S | 必看（高价值） | 红色 |
| A | 常看（稳定价值） | 橙色 |
| B | 偶尔（补充/娱乐） | 蓝色 |
| C | 观察（暂存/可能取关） | 灰色 |
| D | 屏蔽（不推荐） | 黑色 |
