# 视觉设计文档

**项目**：抖音关注博主筛选助手
**版本**：v1.0
**日期**：2024年

---

## 1. 设计规范

### 1.1 设计原则

| 原则 | 描述 |
|------|------|
| 简洁高效 | 减少视觉噪声，突出内容本身 |
| 层次分明 | 信息层级清晰，主次分明 |
| 操作导向 | 视觉引导用户完成操作 |
| 一致性 | 保持全端视觉体验统一 |

### 1.2 设计风格

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              设计风格                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  风格参考: 轻量化工具 + 数据可视化                                          │
│  ────────────────────────────────────────────────────────────────          │
│                                                                             │
│  • 大量留白，减少视觉拥挤                                                  │
│  • 卡片圆角: 12px                                                          │
│  • 图标风格: Linear Icon (线性图标)                                        │
│  • 插画风格: 简约扁平                                                       │
│  • 数据可视化: 清晰直观，避免过度装饰                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. 颜色系统

### 2.1 主色调

| 用途 | 颜色 | 色值 | 使用场景 |
|------|------|------|---------|
| 主色 | 抖音红 | `#FE2C55` | 重点操作、强调元素 |
| 辅助色 | 科技蓝 | `#25F4EE` | 次要操作、辅助信息 |
| 背景色 | 深空黑 | `#1A1A1A` | 主背景 |
| 卡片色 | 炭灰色 | `#252525` | 卡片背景 |
| 边框色 | 浅灰 | `#333333` | 分割线、边框 |

### 2.2 文字色

| 级别 | 颜色 | 色值 | 使用场景 |
|------|------|------|---------|
| 主文字 | 纯白 | `#FFFFFF` | 标题、重要信息 |
| 次文字 | 浅灰 | `#B3B3B3` | 副标题、说明文字 |
| 弱文字 | 深灰 | `#666666` | 辅助信息、时间戳 |
| 禁用文字 | 暗灰 | `#4D4D4D` | 禁用状态 |

### 2.3 优先级颜色

| 优先级 | 颜色 | 色值 | 视觉感受 |
|--------|------|------|---------|
| S级 | 正红色 | `#FF3B30` | 紧急、必看 |
| A级 | 橙色 | `#FF9500` | 重要、常看 |
| B级 | 蓝色 | `#007AFF` | 普通、偶尔 |
| C级 | 灰色 | `#8E8E93` | 观察、暂存 |
| D级 | 深灰 | `#48484A` | 屏蔽、不推荐 |

### 2.4 状态颜色

| 状态 | 颜色 | 色值 | 使用场景 |
|------|------|------|---------|
| 成功 | 绿色 | `#34C759` | 操作成功 |
| 警告 | 橙色 | `#FF9500` | 需要注意 |
| 错误 | 红色 | `#FF3B30` | 操作失败 |
| 信息 | 蓝色 | `#007AFF` | 提示信息 |
