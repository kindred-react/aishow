# 抖音关注博主筛选助手 - 功能清单与迭代路线图

## 一、功能清单与优先级

### 🔴 P0 - MVP 核心（保证可用）

| 功能模块 | 功能点 | 描述 |
|---------|--------|------|
| **导入管理** | 手动导入博主 | 粘贴/录入博主昵称、主页链接、备注 |
| **博主管理** | 博主列表 | 表格/卡片视图展示所有已导入博主 |
| **标签系统** | 手动打标签 | L1分类：学习/资讯/技能/审美/娱乐/情绪/其他 |
| **分层系统** | 手动优先级分层 | S/A/B/C/D 五级分层 |
| **筛选功能** | 多维筛选 | 按领域、评分、更新频率等筛选 |
| **输出功能** | 导出CSV | 分层结果/标签导出 |
| **待办清单** | 每周复盘清单 | 不更新/广告高/重复高 → 待复核 |

### 🟠 P1 - V1（体验完善）

| 功能模块 | 功能点 | 描述 |
|---------|--------|------|
| **规则引擎** | 自动分层规则 | 近30天更新≥8且信息密度≥4 → A以上 |
| **规则引擎** | 规则配置 | 可视化配置条件表达式 |
| **AI辅助** | AI推荐初始评分 | 基于简介/代表内容推荐标签与评分 |
| **AI辅助** | 解释型输出 | 为什么推荐/降权（规则命中与证据） |
| **异常检测** | 账号状态检测 | 识别已注销/不可访问/疑似封禁 |
| **异常处理** | 待确认取关队列 | 进入"待确认取关"清单，用户一键确认 |
| **撤销机制** | 撤销与恢复 | 支持撤销取关操作 |

### 🟡 P2 - V2（智能化）

| 功能模块 | 功能点 | 描述 |
|---------|--------|------|
| **AI能力** | 用户反馈学习 | 用户上调/下调后，规则权重调整 |
| **AI能力** | 视频内容摘要 | 单条视频3-7要点+结论/建议 |
| **AI能力** | 博主维度聚类 | 对最近N条视频聚类，产出主题变化 |
| **差异输出** | 知识类周报 | 本周核心观点/方法、可执行清单 |
| **差异输出** | 娱乐类播放清单 | 按场景+时长自动拼单 |
| **半自动导入** | 分享链接批量导入 | 识别抖音分享链接批量导入 |
| **半自动导入** | 主页链接导入 | 输入博主主页URL自动识别 |

### 🟢 P3 - V3（生态扩展）

| 功能模块 | 功能点 | 描述 |
|---------|--------|------|
| **跨平台** | B站创作者管理 | 同步管理B站博主 |
| **跨平台** | 小红书创作者管理 | 同步管理小红书博主 |
| **策展系统** | Timely内容管理 | 时效性内容生命周期（到期/降权/简报） |
| **策展系统** | Evergreen知识库 | 长期留存内容结构化沉淀 |
| **发现功能** | 优质博主发现 | 根据已标记S/A推荐同赛道高质量创作者 |
| **行为优化** | 时间预算提醒 | 识别沉迷信号，轻量干预 |
| **画像报告** | 画像变化周报 | 本周你给算法的信号偏向哪些赛道 |

## 三、详细迭代计划

### MVP v0.1（2-3周）- 最小可用

**目标**：验证核心流程，跑通手动管理闭环

| 功能 | 描述 | 工时 |
|------|------|------|
| 手动导入博主 | 昵称+链接+备注 | 1天 |
| 博主列表页 | 表格展示，支持排序 | 1天 |
| 标签管理 | L1分类手动打标 | 1天 |
| 优先级分层 | S/A/B/C/D手动分层 | 1天 |
| 筛选功能 | 按标签/层级筛选 | 1天 |
| CSV导出 | 导出全部数据 | 0.5天 |
| 复盘清单 | 按规则生成待复核列表 | 1天 |

**交付物**：手动导入-打标-分层-筛选-导出闭环

---

### MVP v0.2（2-3周）- 体验完善

**目标**：提升管理效率，增加可靠性

| 功能 | 描述 | 工时 |

| 账号状态检测 | 检测已注销/不可访问 | 2天 |
| 待确认取关队列 | 异常账号进队列 | 1天 |
| 规则引擎配置 | 可视化配置分层规则 | 2天 |
| 撤销/恢复机制 | 取关可撤销 | 1天 |
| 批量操作 | 批量打标/分层 | 1天 |
| 数据持久化 | 本地存储/云同步 | 2天 |

**交付物**：异常账号自动识别、规则自动化分层、操作可撤销

## 四、优先级决策

```
用户痛点强度排序：

1. 🔴 关注太多刷不过来          → MVP解决（分层+筛选）
2. 🔴 不知道该看谁               → V1解决（AI评分+自动分层）
3. 🟠 想减掉低质量关注但怕删错    → v0.2解决（异常检测+撤销）
4. 🟠 视频太多时间不够           → V2解决（摘要+播放清单）
5. 🟡 想发现更多优质博主          → V3解决（博主发现）
6. 🟢 跨平台管理麻烦             → V3解决（跨平台）
```
