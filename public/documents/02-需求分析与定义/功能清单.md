## 二、P0 - MVP 核心功能

### 2.1 导入管理

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| 手动导入博主 | 粘贴/录入博主昵称、主页链接、备注 | 基础 | P0 |
| 博主列表展示 | 表格/卡片视图展示所有已导入博主 | 基础 | P0 |

**交互说明**：
- 用户点击"添加博主"按钮
- 弹出表单，填写昵称、主页链接、备注（可选）
- 提交后博主出现在列表中

**验收标准**：
- [ ] 可以成功添加博主
- [ ] 重复链接提示错误
- [ ] 空表单提交时提示必填

---

### 2.2 标签系统

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| L1分类打标 | 手动选择主类目：学习/资讯/技能/审美/娱乐/情绪/其他 | 基础 | P0 |

**L1分类定义**：

| 分类 | 描述 | 示例 |
|------|------|------|
| 知识/学习 | 历史分析、人文讲解、科普、读书、方法论 | 老高小术、所长林超 |
| 资讯/观点 | 商业、科技、热点解读、投资观点 | 商业财经、科技解说 |
| 技能/教程 | 健身训练、英语学习、摄影剪辑、做饭手工 | 健身教程、语言学习 |
| 审美/颜值 | 好看/帅（外貌与氛围为主） | 颜值博主、时尚 |
| 娱乐/抽象 | 搞笑、整活、抽象段子、二创 | 搞笑博主、剧情 |
| 情绪价值 | 治愈、陪伴、共鸣、生活方式 | 生活博主、治愈系 |
| 其他 | 带货导购、直播切片等 | 带货、直播 |

**交互说明**：
- 在博主详情页，点击标签区域
- 弹出分类选择器（单选）
- 选择后自动保存

**验收标准**：
- [ ] 7个分类都可以选择
- [ ] 选择后显示在博主卡片上
- [ ] 可以修改已选分类

---

### 2.3 分层系统

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| 手动优先级分层 | S/A/B/C/D 五级分层 | 基础 | P0 |

**分层定义**：

| 层级 | 描述 | 典型用户意图 |
|------|------|-------------|
| S | 必看（高价值） | 每天必刷 |
| A | 常看（稳定价值） | 有空就看 |
| B | 偶尔（补充/娱乐） | 偶尔看看 |
| C | 观察（暂存/可能取关） | 观望中 |
| D | 屏蔽（不推荐） | 不再看 |

**交互说明**：
- 在博主详情页或列表中，点击优先级标签
- 弹出层级选择器（单选）
- 选择后自动保存

**验收标准**：
- [ ] 5个层级都可以选择
- [ ] 选择后显示在博主卡片上
- [ ] 可以修改已选层级
- [ ] 列表页可以按层级筛选

---

### 2.4 筛选功能

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| 多维筛选 | 按领域、评分、更新频率等筛选 | 基础 | P0 |

**筛选维度**：

| 维度 | 类型 | 描述 |
|------|------|------|
| L1分类 | 单选/多选 | 按分类筛选 |
| 优先级 | 多选 | 按S/A/B/C/D筛选 |
| 更新频率 | 多选 | 按更新频率筛选 |
| 状态 | 多选 | 正常/已注销/不可访问 |
| 关键词 | 文本 | 按昵称搜索 |

**交互说明**：
- 列表页顶部显示筛选栏
- 点击筛选按钮展开筛选面板
- 选择条件后实时筛选结果

**验收标准**：
- [ ] 每个维度都可以筛选
- [ ] 筛选条件可以组合
- [ ] 筛选结果实时更新
- [ ] 显示当前筛选条件

---

### 2.5 输出功能

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| CSV导出 | 分层结果/标签导出 | 基础 | P0 |
| 复盘清单 | 不更新/广告高/重复高 → 待复核 | 基础 | P0 |

**CSV导出**：

| 字段 | 描述 |
|------|------|
| 昵称 | 博主名称 |
| 主页链接 | 抖音主页URL |
| L1分类 | 主类目 |
| 优先级 | S/A/B/C/D |
| 更新频率 | 周更/日更/不稳定/停更 |
| 备注 | 用户备注 |

**交互说明**：
- 点击"导出"按钮
- 选择导出范围（全量/当前筛选）
- 生成CSV文件并下载

**验收标准**：
- [ ] 可以导出全量数据
- [ ] 可以导出筛选后数据
- [ ] CSV格式正确

---

## 三、P1 - V1 增强功能

### 3.1 规则引擎

| 功能点 | 描述 | 类型 | 优先级 |
|-------|------|------|--------|
| 规则配置 | 可视化配置分层规则 | 增强 | P1 |
| 自动分层 | 按规则自动分层 | 增强 | P1 |

**规则示例**：

```
IF 近30天更新 >= 8 AND 信息密度 >= 4
THEN 建议 >= A

IF 广告占比 >= 4 OR 重复度 >= 4
THEN 降 1-2 级
```

**验收标准**：
- [ ] 可以添加自定义规则
- [ ] 规则可以启用/禁用
- [ ] 规则变更有日志记录
