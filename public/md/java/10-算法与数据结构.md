## 10.1 复杂度分析

### 10.1.1 时间复杂度

**时间复杂度**（Time Complexity）用大 O 表示法描述算法运行时间与输入规模 n 的增长关系，反映算法的**渐进时间复杂度**。

#### 常见时间复杂度

| 复杂度 | 名称 | 典型算法 | n=10 | n=100 | n=1000 |
|--------|------|----------|------|-------|--------|
| **O(1)** | 常数 | 数组随机访问 | 1 | 1 | 1 |
| **O(log n)** | 对数 | 二分查找 | 3 | 7 | 10 |
| **O(n)** | 线性 | 线性查找 | 10 | 100 | 1000 |
| **O(n log n)** | 线性对数 | 归并排序 | 33 | 664 | 9966 |
| **O(n^2)** | 平方 | 冒泡排序 | 100 | 10000 | 10^6 |
| **O(2^n)** | 指数 | 子集枚举 | 1024 | 10^30 | 10^300 |
| **O(n!)** | 阶乘 | 全排列 | 3628800 | 极大 | 极大 |

#### 复杂度增长趋势（从小到大）

```
O(1) < O(log n) < O(n) < O(n log n) < O(n^2) < O(n^3) < O(2^n) < O(n!)
```

#### Java 代码示例

```java
// O(1) - 常数时间
int getFirst(int[] arr) {
    return arr[0];
}

// O(log n) - 对数时间（二分查找）
int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// O(n) - 线性时间
int sum(int[] arr) {
    int total = 0;
    for (int num : arr) total += num;
    return total;
}

// O(n log n) - 线性对数（归并排序核心）
void mergeSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    mergeSort(arr, left, mid);      // O(log n) 层递归
    mergeSort(arr, mid + 1, right);  // 每层 O(n) 合并
    merge(arr, left, mid, right);
}

// O(n^2) - 平方时间（冒泡排序）
void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length; i++) {          // O(n)
        for (int j = 0; j < arr.length - 1 - i; j++) { // O(n)
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

// O(2^n) - 指数时间（斐波那契递归）
int fib(int n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2); // 两个递归调用，形成二叉树
}
```

### 10.1.2 空间复杂度

**空间复杂度**（Space Complexity）描述算法执行过程中**额外**使用的内存空间与输入规模 n 的关系。

```java
// O(1) 空间 - 原地操作
void reverse(int[] arr) {
    int left = 0, right = arr.length - 1;
    while (left < right) {
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        left++; right--;
    }
}

// O(n) 空间 - 需要额外数组
int[] copyArray(int[] arr) {
    int[] copy = new int[arr.length]; // 额外 O(n) 空间
    System.arraycopy(arr, 0, copy, 0, arr.length);
    return copy;
}

// O(n) 空间 - 递归调用栈
int fibRecursive(int n) {
    if (n <= 1) return n;
    return fibRecursive(n - 1) + fibRecursive(n - 2);
    // 最深递归 n 层，每层一个栈帧 → O(n) 空间
}
```

### 10.1.3 最好/最坏/平均/均摊复杂度

```java
// 在数组中查找元素 x
int find(int[] arr, int x) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == x) return i;
    }
    return -1;
}
```

| 复杂度类型 | 含义 | 上例分析 |
|-----------|------|---------|
| **最好情况** | 最理想的输入下的运行时间 | x 在第一个位置 → **O(1)** |
| **最坏情况** | 最不理想的输入下的运行时间 | x 不存在或最后位置 → **O(n)** |
| **平均情况** | 所有可能输入的期望运行时间 | x 在每个位置概率相同 → **O(n)** |
| **均摊复杂度** | 一系列操作的总复杂度均摊到每次操作 | 如 ArrayList 扩容 → 均摊 **O(1)** |

#### 均摊复杂度详解（ArrayList 扩容）

```java
// ArrayList.add() 的均摊分析
// 假设初始容量为 1，每次满时扩容为 2 倍
// 插入 n 个元素的总扩容复制次数：
// 1 + 2 + 4 + 8 + ... + n ≈ 2n
// 总操作次数 = n（插入）+ 2n（复制）= 3n
// 均摊到每次插入：3n / n = O(1)
```

### 10.1.4 常见复杂度对比表

| 复杂度 | 10次操作 | 100次操作 | 1000次操作 | 可行性 |
|--------|---------|----------|-----------|--------|
| O(1) | 1 | 1 | 1 | 极优 |
| O(log n) | 3 | 7 | 10 | 优秀 |
| O(n) | 10 | 100 | 1,000 | 良好 |
| O(n log n) | 33 | 664 | 9,966 | 可接受 |
| O(n^2) | 100 | 10,000 | 1,000,000 | 勉强 |
| O(2^n) | 1,024 | 1.27x10^30 | 不可计算 | 不可行 |
| O(n!) | 3,628,800 | 不可计算 | 不可计算 | 不可行 |

> **面试要点**：分析复杂度时关注**最高阶项**，忽略常数系数和低阶项。例如 `3n^2 + 5n + 100 = O(n^2)`。

## 10.2 数据结构

### 10.2.1 数组与链表

#### 数组 vs 链表对比

| 特性 | 数组 | 链表 |
|------|------|------|
| **内存分配** | 连续内存空间 | 离散内存空间 |
| **随机访问** | O(1) | O(n) |
| **头部插入** | O(n) | O(1) |
| **尾部插入** | O(1) 均摊 | O(n)（无尾指针）/ O(1) |
| **中间插入** | O(n) | O(1)（已知位置） |
| **查找** | O(n) / O(log n)（有序+二分） | O(n) |
| **空间** | 紧凑，无额外开销 | 每个节点额外存指针 |
| **缓存友好性** | 高（CPU 缓存命中率高） | 低（内存不连续） |
| **扩容** | 需要重新分配 | 动态增长 |

#### 单链表实现

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int val) { this.val = val; }
}

// 头插法
ListNode insertAtHead(ListNode head, int val) {
    ListNode node = new ListNode(val);
    node.next = head;
    return node;
}

// 尾插法
ListNode insertAtTail(ListNode head, int val) {
    ListNode node = new ListNode(val);
    if (head == null) return node;
    ListNode cur = head;
    while (cur.next != null) cur = cur.next;
    cur.next = node;
    return head;
}

// 反转链表（迭代）
ListNode reverseList(ListNode head) {
    ListNode prev = null, cur = head;
    while (cur != null) {
        ListNode next = cur.next;
        cur.next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
}

// 反转链表（递归）
ListNode reverseListRecursive(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = reverseListRecursive(head.next);
    head.next.next = head;
    head.next = null;
    return newHead;
}
```

#### 双向链表实现

```java
class DoublyListNode {
    int val;
    DoublyListNode prev, next;
    DoublyListNode(int val) { this.val = val; }
}

// 双向链表插入
DoublyListNode insertAfter(DoublyListNode node, int val) {
    DoublyListNode newNode = new DoublyListNode(val);
    newNode.prev = node;
    newNode.next = node.next;
    if (node.next != null) node.next.prev = newNode;
    node.next = newNode;
    return newNode;
}

// 双向链表删除
DoublyListNode deleteNode(DoublyListNode node) {
    if (node.prev != null) node.prev.next = node.next;
    if (node.next != null) node.next.prev = node.prev;
    return node.next;
}
```

#### 循环链表

```java
// 判断链表是否有环（快慢指针）
boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) return false;
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;          // 慢指针走一步
        fast = fast.next.next;     // 快指针走两步
        if (slow == fast) return true; // 相遇则有环
    }
    return false;
}

// 找到环的入口节点
ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) {
            // 快指针从头开始，慢指针继续，再次相遇即为入口
            ListNode ptr1 = head, ptr2 = slow;
            while (ptr1 != ptr2) {
                ptr1 = ptr1.next;
                ptr2 = ptr2.next;
            }
            return ptr1;
        }
    }
    return null;
}
```

#### ArrayList vs LinkedList 源码分析

**ArrayList 核心原理**：

```java
public class ArrayList<E> extends AbstractList<E> {
    private static final int DEFAULT_CAPACITY = 10;
    transient Object[] elementData;  // 底层数组
    private int size;                // 实际元素个数

    // add() - 时间复杂度均摊 O(1)
    public boolean add(E e) {
        ensureCapacityInternal(size + 1); // 检查是否需要扩容
        elementData[size++] = e;
        return true;
    }

    // 扩容机制 - 1.5 倍扩容
    private void grow(int minCapacity) {
        int oldCapacity = elementData.length;
        int newCapacity = oldCapacity + (oldCapacity >> 1); // oldCapacity * 1.5
        if (newCapacity < minCapacity)
            newCapacity = minCapacity;
        elementData = Arrays.copyOf(elementData, newCapacity);
    }

    // get() - O(1)
    public E get(int index) {
        rangeCheck(index);
        return elementData(index);
    }

    // remove() - O(n)，需要移动元素
    public E remove(int index) {
        E oldValue = elementData(index);
        int numMoved = size - index - 1;
        if (numMoved > 0)
            System.arraycopy(elementData, index + 1, elementData, index, numMoved);
        elementData[--size] = null; // help GC
        return oldValue;
    }
}
```

**LinkedList 核心原理**：

```java
public class LinkedList<E> extends AbstractSequentialList<E>
        implements List<E>, Deque<E> {
    transient Node<E> first; // 头节点
    transient Node<E> last;  // 尾节点
    transient int size = 0;

    private static class Node<E> {
        E item;
        Node<E> next;
        Node<E> prev;
        Node(Node<E> prev, E element, Node<E> next) {
            this.item = element;
            this.next = next;
            this.prev = prev;
        }
    }

    // 头部插入 - O(1)
    private void linkFirst(E e) {
        final Node<E> f = first;
        final Node<E> newNode = new Node<>(null, e, f);
        first = newNode;
        if (f == null) last = newNode;
        else f.prev = newNode;
        size++;
    }

    // 尾部插入 - O(1)
    void linkLast(E e) {
        final Node<E> l = last;
        final Node<E> newNode = new Node<>(l, e, null);
        last = newNode;
        if (l == null) first = newNode;
        else l.next = newNode;
        size++;
    }

    // get() - O(n)，需要遍历
    public E get(int index) {
        checkElementIndex(index);
        return node(index).item;
    }

    // 优化：根据索引位置决定从头还是从尾遍历
    Node<E> node(int index) {
        if (index < (size >> 1)) {
            Node<E> x = first;
            for (int i = 0; i < index; i++) x = x.next;
            return x;
        } else {
            Node<E> x = last;
            for (int i = size - 1; i > index; i--) x = x.prev;
            return x;
        }
    }
}
```

> **面试要点**：ArrayList 在大多数场景下性能优于 LinkedList，因为 CPU 缓存对连续内存更友好。LinkedList 的理论优势（O(1) 插入删除）在实际中往往被遍历开销抵消。

### 10.2.2 栈与队列

#### 栈的应用

**括号匹配**：

```java
boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            stack.push(c);
        } else {
            if (stack.isEmpty()) return false;
            char top = stack.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
    }
    return stack.isEmpty();
}
```

**表达式求值（后缀表达式 / 逆波兰表达式）**：

```java
int evalRPN(String[] tokens) {
    Deque<Integer> stack = new ArrayDeque<>();
    for (String token : tokens) {
        switch (token) {
            case "+": stack.push(stack.pop() + stack.pop()); break;
            case "-": {
                int b = stack.pop(), a = stack.pop();
                stack.push(a - b); break;
            }
            case "*": stack.push(stack.pop() * stack.pop()); break;
            case "/": {
                int b = stack.pop(), a = stack.pop();
                stack.push(a / b); break;
            }
            default: stack.push(Integer.parseInt(token));
        }
    }
    return stack.pop();
}
```

**单调栈（下一个更大元素）**：

```java
int[] nextGreaterElement(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    Arrays.fill(result, -1);
    Deque<Integer> stack = new ArrayDeque<>(); // 存索引

    for (int i = 0; i < n; i++) {
        while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
            result[stack.pop()] = nums[i];
        }
        stack.push(i);
    }
    return result;
}
```

#### 队列

**循环队列实现**：

```java
class CircularQueue {
    private int[] data;
    private int head, tail, size, capacity;

    public CircularQueue(int k) {
        data = new int[k];
        capacity = k;
        head = 0; tail = 0; size = 0;
    }

    public boolean enQueue(int value) {
        if (isFull()) return false;
        data[tail] = value;
        tail = (tail + 1) % capacity;
        size++;
        return true;
    }

    public boolean deQueue() {
        if (isEmpty()) return false;
        head = (head + 1) % capacity;
        size--;
        return true;
    }

    public int Front() { return isEmpty() ? -1 : data[head]; }
    public int Rear() { return isEmpty() ? -1 : data[(tail - 1 + capacity) % capacity]; }
    public boolean isEmpty() { return size == 0; }
    public boolean isFull() { return size == capacity; }
}
```

#### Java 中 Stack / Deque / PriorityQueue

| 类/接口 | 底层实现 | 特点 | 推荐度 |
|---------|---------|------|--------|
| **Stack** | 数组 | 线程安全，继承 Vector，性能较差 | 不推荐 |
| **ArrayDeque** | 数组（循环数组） | 非线程安全，性能好 | **推荐** |
| **LinkedList** | 双向链表 | 非线程安全，同时实现 List 和 Deque | 适中 |
| **PriorityQueue** | 堆（数组实现） | 非线程安全，默认最小堆 | 适合堆场景 |

> **面试要点**：Java 官方推荐使用 `Deque` 接口替代 `Stack` 类。`ArrayDeque` 作为栈使用时比 `Stack` 性能更好。

### 10.2.3 哈希表

#### 哈希函数设计

**好的哈希函数应满足**：
1. **确定性**：相同输入始终产生相同输出
2. **均匀性**：减少哈希冲突
3. **高效性**：计算速度快

```java
// Java String.hashCode() 实现
public int hashCode() {
    int h = 0;
    for (int i = 0; i < value.length; i++) {
        h = 31 * h + value[i]; // 乘以 31（质数，减少冲突）
    }
    return h;
}

// HashMap 中的哈希扰动函数
static final int hash(Object key) {
    int h;
    return (key == null) ? 0 : (h = key.hashCode()) ^ (h >>> 16);
    // 高16位与低16位异或，增加散列性
}
```

#### 哈希冲突解决

**拉链法（Separate Chaining）**：

```java
// 简单的拉链法哈希表
class MyHashMap {
    private static final int SIZE = 16;
    private LinkedList<Entry>[] buckets;

    private static class Entry {
        int key, value;
        Entry(int key, int value) { this.key = key; this.value = value; }
    }

    public MyHashMap() {
        buckets = new LinkedList[SIZE];
        for (int i = 0; i < SIZE; i++) buckets[i] = new LinkedList<>();
    }

    public void put(int key, int value) {
        int index = key % SIZE;
        for (Entry e : buckets[index]) {
            if (e.key == key) { e.value = value; return; }
        }
        buckets[index].add(new Entry(key, value));
    }

    public int get(int key) {
        int index = key % SIZE;
        for (Entry e : buckets[index]) {
            if (e.key == key) return e.value;
        }
        return -1;
    }
}
```

**开放寻址法（Open Addressing）**：

```java
// 线性探测
class OpenAddressingHashMap {
    private int[] keys;
    private int[] values;
    private static final int DELETED = -1;

    public void put(int key, int value) {
        int index = key % keys.length;
        while (keys[index] != 0 && keys[index] != DELETED) {
            if (keys[index] == key) { values[index] = value; return; }
            index = (index + 1) % keys.length; // 线性探测
        }
        keys[index] = key;
        values[index] = value;
    }
}
```

#### HashMap 源码分析（JDK 8）

**核心数据结构**：数组 + 链表 + 红黑树

```java
public class HashMap<K, V> {
    // 默认初始容量 16
    static final int DEFAULT_INITIAL_CAPACITY = 1 << 4;
    // 最大容量
    static final int MAXIMUM_CAPACITY = 1 << 30;
    // 默认负载因子 0.75
    static final float DEFAULT_LOAD_FACTOR = 0.75f;
    // 链表转红黑树阈值
    static final int TREEIFY_THRESHOLD = 8;
    // 红黑树转链表阈值
    static final int UNTREEIFY_THRESHOLD = 6;
    // 哈希表数组
    transient Node<K, V>[] table;

    // 节点结构（链表节点）
    static class Node<K, V> implements Map.Entry<K, V> {
        final int hash;
        final K key;
        V value;
        Node<K, V> next;
    }

    // 红黑树节点
    static final class TreeNode<K, V> extends Node<K, V> {
        TreeNode<K, V> parent;
        TreeNode<K, V> left, right;
        boolean red;
    }

    // put 操作核心流程
    public V put(K key, V value) {
        return putVal(hash(key), key, value, false, true);
    }

    final V putVal(int hash, K key, V value, boolean onlyIfAbsent, boolean evict) {
        Node<K, V>[] tab; Node<K, V> p; int n, i;
        // 1. 表为空则初始化
        if ((tab = table) == null || (n = tab.length) == 0)
            n = (tab = resize()).length;
        // 2. 计算索引位置，为空则直接插入
        if ((p = tab[i = (n - 1) & hash]) == null)
            tab[i] = newNode(hash, key, value, null);
        else {
            Node<K, V> e; K k;
            // 3. key 相同则覆盖
            if (p.hash == hash && ((k = p.key) == key || (key != null && key.equals(k))))
                e = p;
            // 4. 红黑树节点
            else if (p instanceof TreeNode)
                e = ((TreeNode<K, V>) p).putTreeVal(this, tab, hash, key, value);
            // 5. 链表处理
            else {
                for (int binCount = 0; ; binCount++) {
                    if ((e = p.next) == null) {
                        p.next = newNode(hash, key, value, null);
                        // 链表长度 >= 8 转红黑树
                        if (binCount >= TREEIFY_THRESHOLD - 1)
                            treeifyBin(tab, hash);
                        break;
                    }
                    if (e.hash == hash && ((k = e.key) == key || (key != null && key.equals(k))))
                        break;
                    p = e;
                }
            }
            if (e != null) {
                V oldValue = e.value;
                if (!onlyIfAbsent || oldValue == null)
                    e.value = value;
                return oldValue;
            }
        }
        // 6. 超过阈值则扩容
        if (++size > threshold)
            resize();
        return null;
    }
}
```

#### HashMap 扩容机制

```java
final Node<K, V>[] resize() {
    Node<K, V>[] oldTab = table;
    int oldCap = (oldTab == null) ? 0 : oldTab.length;
    int oldThr = threshold;
    int newCap, newThr = 0;

    // 计算新容量
    if (oldCap > 0) {
        if (oldCap >= MAXIMUM_CAPACITY) {
            threshold = Integer.MAX_VALUE;
            return oldTab;
        }
        newCap = oldCap << 1; // 2 倍扩容
        if (newCap < MAXIMUM_CAPACITY && oldCap >= DEFAULT_INITIAL_CAPACITY)
            newThr = oldThr << 1; // 阈值也翻倍
    }

    // 重新分配元素
    Node<K, V>[] newTab = (Node<K, V>[]) new Node[newCap];
    table = newTab;

    if (oldTab != null) {
        for (int j = 0; j < oldCap; ++j) {
            Node<K, V> e;
            if ((e = oldTab[j]) != null) {
                oldTab[j] = null;
                if (e.next == null)
                    // 元素在新数组中的位置：原位置 或 原位置 + oldCap
                    newTab[e.hash & (newCap - 1)] = e;
                else if (e instanceof TreeNode)
                    ((TreeNode<K, V>) e).split(this, newTab, j, oldCap);
                else {
                    // 链表拆分：根据 (hash & oldCap) 是否为 0 分成两段
                    Node<K, V> loHead = null, loTail = null;
                    Node<K, V> hiHead = null, hiTail = null;
                    do {
                        if ((e.hash & oldCap) == 0) {
                            if (loTail == null) loHead = e;
                            else loTail.next = e;
                            loTail = e;
                        } else {
                            if (hiTail == null) hiHead = e;
                            else hiTail.next = e;
                            hiTail = e;
                        }
                    } while ((e = e.next) != null);
                    if (loTail != null) { loTail.next = null; newTab[j] = loHead; }
                    if (hiTail != null) { hiTail.next = null; newTab[j + oldCap] = hiHead; }
                }
            }
        }
    }
    return newTab;
}
```

> **面试要点**：
> - JDK 8 的 HashMap 由 **数组 + 链表 + 红黑树** 组成
> - 链表长度 >= 8 且数组长度 >= 64 时转红黑树
> - 扩容为 **2 倍**，元素位置要么不变，要么移动到 `原位置 + oldCap`
> - 负载因子默认 **0.75**，是时间和空间的平衡

```java
List<List<Integer>> levelOrder(TreeNode root) {

    if (root == null) return result;
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
```

#### 二叉搜索树（BST）

**性质**：左子树所有节点值 < 根节点值 < 右子树所有节点值

```java
TreeNode searchBST(TreeNode root, int val) {
    if (root == null || root.val == val) return root;
    if (val < root.val) return searchBST(root.left, val);
    return searchBST(root.right, val);
}

// BST 插入
TreeNode insertIntoBST(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);
    if (val < root.val) root.left = insertIntoBST(root.left, val);
    else root.right = insertIntoBST(root.right, val);
    return root;
}

// BST 删除
TreeNode deleteNode(TreeNode root, int key) {
    if (root == null) return null;
    if (key < root.val) {
        root.left = deleteNode(root.left, key);
    } else if (key > root.val) {
        root.right = deleteNode(root.right, key);
    } else {
        // 找到要删除的节点
        if (root.left == null) return root.right;
        if (root.right == null) return root.left;
        // 有两个子节点：用右子树最小值替换
        TreeNode minNode = findMin(root.right);
        root.val = minNode.val;
        root.right = deleteNode(root.right, minNode.val);
    }
    return root;
}

TreeNode findMin(TreeNode node) {
    while (node.left != null) node = node.left;
    return node;
}

boolean isValidBST(TreeNode root) {
    return isValidBST(root, Long.MIN_VALUE, Long.MAX_VALUE);
}

boolean isValidBST(TreeNode node, long min, long max) {
    if (node == null) return true;
    if (node.val <= min || node.val >= max) return false;
    return isValidBST(node.left, min, node.val) && isValidBST(node.right, node.val, max);
}
```

#### AVL 树与红黑树

**AVL 树**：
- **严格平衡**：任意节点的左右子树高度差不超过 1
- 查找效率高，但插入/删除时旋转次数多
- 适用于**查找密集**的场景

**红黑树**：
- **近似平衡**：最长路径不超过最短路径的 2 倍
- 插入/删除时旋转次数少
- JDK 8 的 `TreeMap`、`TreeSet`、`HashMap`（链表转树）使用红黑树

| 特性 | AVL 树 | 红黑树 |
|------|--------|--------|
| **平衡标准** | 严格（|高度差| <= 1） | 宽松（最长 <= 2 * 最短） |
| **查找** | O(log n)，更快 | O(log n) |
| **插入/删除** | 旋转较多 | 旋转较少（最多 3 次） |
| **适用场景** | 查找密集 | 插入/删除频繁 |

**红黑树五大性质**：
1. 每个节点是**红色**或**黑色**
2. **根节点**是黑色
3. 每个**叶子节点**（NIL）是黑色
4. 红色节点的两个子节点都是黑色（**不能有连续红节点**）
5. 从任一节点到其每个叶子的所有路径都包含**相同数目的黑色节点**

#### B 树与 B+ 树（MySQL 索引原理）

**B 树**：
- 多路平衡查找树，每个节点可以有多个子节点
- 所有叶子节点在同一层
- 数据存储在所有节点中

**B+ 树**（MySQL InnoDB 使用）：
- 非叶子节点只存**索引**（键值），不存数据
- 所有数据存储在**叶子节点**
- 叶子节点通过**链表**连接，支持范围查询
- 每个叶子节点到根的距离相同

```
B+ 树结构示意：
              [30 | 60]
            /     |     \
       [10,20]  [40,50]  [70,80,90]
         |        |          |
      [数据]    [数据]    [数据] ← 叶子节点链表连接
```

| 特性 | B 树 | B+ 树 |
|------|------|-------|
| **数据存储** | 所有节点 | 仅叶子节点 |
| **范围查询** | 需要中序遍历 | 叶子链表直接遍历 |
| **单次查询** | 可能提前结束 | 必须到叶子节点 |
| **磁盘 I/O** | 较多 | 较少（非叶子节点更小） |

> **面试要点**：MySQL InnoDB 使用 B+ 树作为索引结构，因为 B+ 树的非叶子节点不存数据，一个节点可以容纳更多索引项，减少树的高度和磁盘 I/O 次数。B+ 树的叶子节点链表支持高效的范围查询。

```java
            if (node.children[c - 'a'] == null) {
                node.children[c - 'a'] = new TrieNode();
            }
            node = node.children[c - 'a'];
        }
        node.isEnd = true;

    // 搜索 - O(m)
    public boolean search(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            if (node.children[c - 'a'] == null) return false;
            node = node.children[c - 'a'];
        }
        return node.isEnd;
    }

    public boolean startsWith(String prefix) {
        TrieNode node = root;
        for (char c : prefix.toCharArray()) {
            if (node.children[c - 'a'] == null) return false;
            node = node.children[c - 'a'];
        }
        return true;
    }
}
```

#### 并查集（Union-Find）

```java
class UnionFind {
    private int[] parent;
    private int[] rank;   // 秩（树的高度近似）
    private int count;    // 连通分量数

    public UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        count = n;
        for (int i = 0; i < n; i++) {
            parent[i] = i;
            rank[i] = 1;
        }
    }

    // 查找（带路径压缩）- 近似 O(1)
    public int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]); // 路径压缩
        }
        return parent[x];
    }

    // 合并（按秩合并）- 近似 O(1)
    public boolean union(int x, int y) {
        int rootX = find(x), rootY = find(y);
        if (rootX == rootY) return false;
        // 按秩合并：小树挂到大树下
        if (rank[rootX] > rank[rootY]) {
            parent[rootY] = rootX;
        } else if (rank[rootX] < rank[rootY]) {
            parent[rootX] = rootY;
        } else {
            parent[rootY] = rootX;
            rank[rootX]++;
        }
        count--;
        return true;
    }

    public boolean connected(int x, int y) {
        return find(x) == find(y);
    }

    public int getCount() { return count; }
}
```

```java
void bfs(GraphList graph, int start) {
    int n = graph.vertexCount;
    boolean[] visited = new boolean[n];
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start);
    visited[start] = true;

    while (!queue.isEmpty()) {
        int node = queue.poll();
        System.out.print(node + " ");
        for (int[] edge : graph.adjList.get(node)) {
            int neighbor = edge[0];
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.offer(neighbor);
            }
        }
    }
}

// DFS（深度优先搜索）- 递归
void dfs(GraphList graph, int node, boolean[] visited) {
    visited[node] = true;
    System.out.print(node + " ");
    for (int[] edge : graph.adjList.get(node)) {
        int neighbor = edge[0];
        if (!visited[neighbor]) {
            dfs(graph, neighbor, visited);
        }
    }
}

// DFS - 迭代（使用栈）
void dfsIterative(GraphList graph, int start) {
    int n = graph.vertexCount;
    boolean[] visited = new boolean[n];
    Deque<Integer> stack = new ArrayDeque<>();
    stack.push(start);

    while (!stack.isEmpty()) {
        int node = stack.pop();
        if (visited[node]) continue;
        visited[node] = true;
        System.out.print(node + " ");
        // 注意：逆序压栈以保持正序访问
        List<int[]> neighbors = graph.adjList.get(node);
        for (int i = neighbors.size() - 1; i >= 0; i--) {
            if (!visited[neighbors.get(i)[0]]) {
                stack.push(neighbors.get(i)[0]);
            }
        }
    }
}
```

```java
// Kahn 算法（BFS 实现）
int[] topologicalSort(int numCourses, int[][] prerequisites) {
    List<List<Integer>> graph = new ArrayList<>();
    int[] inDegree = new int[numCourses];
    for (int i = 0; i < numCourses; i++) graph.add(new ArrayList<>());

    for (int[] pre : prerequisites) {
        graph.get(pre[1]).add(pre[0]);
        inDegree[pre[0]]++;
    }

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < numCourses; i++) {
        if (inDegree[i] == 0) queue.offer(i);
    }

    int[] result = new int[numCourses];
    int index = 0;
    while (!queue.isEmpty()) {
        int node = queue.poll();
        result[index++] = node;
        for (int neighbor : graph.get(node)) {
            if (--inDegree[neighbor] == 0) {
                queue.offer(neighbor);
            }
        }
    }

    return index == numCourses ? result : new int[0]; // 有环则返回空
}
```

#### 最短路径

**Dijkstra 算法**（单源最短路径，不能有负权边）：

```java
int[] dijkstra(int[][] graph, int src) {
    int n = graph.length;
    int[] dist = new int[n];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[src] = 0;
    boolean[] visited = new boolean[n];

    // 优先队列：[距离, 节点]
    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
    pq.offer(new int[]{0, src});

    while (!pq.isEmpty()) {
        int[] cur = pq.poll();
        int d = cur[0], u = cur[1];
        if (visited[u]) continue;
        visited[u] = true;

        for (int v = 0; v < n; v++) {
            if (graph[u][v] > 0 && !visited[v] && dist[u] + graph[u][v] < dist[v]) {
                dist[v] = dist[u] + graph[u][v];
                pq.offer(new int[]{dist[v], v});
            }
        }
    }
    return dist;
}
// 时间复杂度：O((V + E) log V)（使用优先队列）
```

**Floyd 算法**（多源最短路径）：

```java
void floyd(int[][] graph) {
    int n = graph.length;
    int[][] dist = new int[n][n];
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            dist[i][j] = graph[i][j];
        }
    }

    // dp[k][i][j] = min(dp[k-1][i][j], dp[k-1][i][k] + dp[k-1][k][j])
    for (int k = 0; k < n; k++) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (dist[i][k] != Integer.MAX_VALUE && dist[k][j] != Integer.MAX_VALUE) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
    }
}
// 时间复杂度：O(V^3)，空间复杂度：O(V^2)
```

#### 最小生成树

**Kruskal 算法**（按边贪心）：

```java
int kruskal(int n, int[][] edges) {
    // 按权重排序
    Arrays.sort(edges, (a, b) -> a[2] - b[2]);
    UnionFind uf = new UnionFind(n);
    int mstWeight = 0, edgeCount = 0;

    for (int[] edge : edges) {
        int u = edge[0], v = edge[1], w = edge[2];
        if (uf.union(u, v)) {
            mstWeight += w;
            edgeCount++;
            if (edgeCount == n - 1) break;
        }
    }
    return edgeCount == n - 1 ? mstWeight : -1;
}
// 时间复杂度：O(E log E)
```

**Prim 算法**（按点扩展）：

```java
int prim(int[][] graph) {
    int n = graph.length;
    int[] dist = new int[n];      // 到 MST 的最短距离
    boolean[] inMST = new boolean[n];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[0] = 0;
    int mstWeight = 0;

    for (int i = 0; i < n; i++) {
        // 找到不在 MST 中距离最小的节点
        int u = -1;
        for (int j = 0; j < n; j++) {
            if (!inMST[j] && (u == -1 || dist[j] < dist[u])) u = j;
        }
        inMST[u] = true;
        mstWeight += dist[u];

        // 更新邻接节点距离
        for (int v = 0; v < n; v++) {
            if (graph[u][v] > 0 && !inMST[v] && graph[u][v] < dist[v]) {
                dist[v] = graph[u][v];
            }
        }
    }
    return mstWeight;
}
// 时间复杂度：O(V^2)，使用优先队列可优化到 O((V + E) log V)
```

```java
void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        boolean swapped = false;
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) break; // 优化：已有序则提前退出
    }
}
// 时间：O(n^2)，空间：O(1)，稳定
```

### 10.3.2 高级排序

#### 希尔排序

```java
void shellSort(int[] arr) {
    int n = arr.length;
    // 间隔序列：n/2, n/4, ..., 1
    for (int gap = n / 2; gap > 0; gap /= 2) {
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j = i;
            while (j >= gap && arr[j - gap] > temp) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            arr[j] = temp;
        }
    }
}
// 时间：O(n log n) ~ O(n^2)，取决于间隔序列，空间：O(1)，不稳定
```

```java
    mergeSort(arr, left, mid, temp);
    mergeSort(arr, mid + 1, right, temp);

void merge(int[] arr, int left, int mid, int right, int[] temp) {
    int i = left, j = mid + 1, k = 0;
    while (i <= mid && j <= right) {
        if (arr[i] <= arr[j]) temp[k++] = arr[i++];
        else temp[k++] = arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    System.arraycopy(temp, 0, arr, left, k);
}
// 时间：O(n log n)，空间：O(n)，稳定
```

```java
void quickSort(int[] arr, int left, int right) {

    quickSort(arr, left, pivotIndex - 1);
    quickSort(arr, pivotIndex + 1, right);
}

int partition(int[] arr, int left, int right) {
    int pivot = arr[right]; // 选最后一个元素为基准
    int i = left;
    for (int j = left; j < right; j++) {
        if (arr[j] < pivot) {
            int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
            i++;
        }
    }
    int temp = arr[i]; arr[i] = arr[right]; arr[right] = temp;
    return i;
}
// 时间：平均 O(n log n)，最坏 O(n^2)，空间：O(log n)（递归栈），不稳定
```

#### 堆排序

```java
void heapSort(int[] arr) {
    int n = arr.length;
    // 建堆 - O(n)
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    // 排序 - O(n log n)
    for (int i = n - 1; i > 0; i--) {
        int temp = arr[0]; arr[0] = arr[i]; arr[i] = temp;
        heapify(arr, i, 0);
    }
}

void heapify(int[] arr, int n, int i) {
    int largest = i;
    int left = 2 * i + 1, right = 2 * i + 2;
    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;
    if (largest != i) {
        int temp = arr[i]; arr[i] = arr[largest]; arr[largest] = temp;
        heapify(arr, n, largest);
    }
}
// 时间：O(n log n)，空间：O(1)，不稳定
```

```java
int[] countingSort(int[] arr) {
    if (arr == null || arr.length == 0) return arr;
    int max = Arrays.stream(arr).max().getAsInt();
    int min = Arrays.stream(arr).min().getAsInt();
    int range = max - min + 1;
    int[] count = new int[range];
    int[] result = new int[arr.length];

    // 累加（计算每个元素的最终位置）

        result[--count[arr[i] - min]] = arr[i];

    return result;
}
```

```java
// 1. 基本类型（int, double 等）→ DualPivotQuicksort（双轴快速排序）

// - 选取两个 pivot（p1, p2），将数组分为三部分：< p1, p1~p2, > p2
// - 对三部分分别递归排序
// - 比经典快排减少了递归深度和比较次数

// - 找到数组中已经有序的连续段（称为 run）
// - 对较短的 run 用插入排序扩展到最小长度（minrun，通常 32~64）
// - 用归并排序将 run 两两合并
// - 利用 galloping 模式优化合并效率
// 时间复杂度：O(n log n)，最好 O(n)
```

### 10.3.6 快排优化

```java
// 优化1：三数取中法选 pivot
int medianOfThree(int[] arr, int left, int right) {
    int mid = left + (right - left) / 2;
    // 确保 arr[left] <= arr[mid] <= arr[right]
    if (arr[left] > arr[mid]) swap(arr, left, mid);
    if (arr[mid] > arr[right]) swap(arr, mid, right);
    if (arr[left] > arr[mid]) swap(arr, left, mid);
    return mid;
}

// 优化2：随机化 pivot
int randomizedPartition(int[] arr, int left, int right) {
    int randomIdx = left + new Random().nextInt(right - left + 1);
    swap(arr, randomIdx, right);
    return partition(arr, left, right);
}

// 优化3：小数组切换插入排序
void quickSortOptimized(int[] arr, int left, int right) {
    if (right - left < 16) {
        insertionSort(arr, left, right); // 小区间用插入排序
        return;
    }
    int pivotIdx = medianOfThree(arr, left, right);
    swap(arr, pivotIdx, right);
    int p = partition(arr, left, right);
    quickSortOptimized(arr, left, p - 1);
    quickSortOptimized(arr, p + 1, right);
}

// 优化4：三路快排（处理大量重复元素）
void quickSort3Way(int[] arr, int left, int right) {
    if (left >= right) return;
    int pivot = arr[left];
    int lt = left, gt = right, i = left + 1;
    // arr[left..lt-1] < pivot, arr[lt..gt] == pivot, arr[gt+1..right] > pivot
    while (i <= gt) {
        if (arr[i] < pivot) swap(arr, lt++, i++);
        else if (arr[i] > pivot) swap(arr, i, gt--);
        else i++;
    }
    quickSort3Way(arr, left, lt - 1);
    quickSort3Way(arr, gt + 1, right);
}
```

### 10.4.4 分块查找

```java
// 将数组分为若干块，每块内无序但块间有序
// 先在索引表中二分查找确定块，再在块内顺序查找
class BlockSearch {
    private int[] arr;
    private int blockSize;
    private int[] maxValues; // 每块最大值（索引表）

    public BlockSearch(int[] arr, int blockSize) {
        this.arr = arr;
        this.blockSize = blockSize;
        int blockCount = (arr.length + blockSize - 1) / blockSize;
        maxValues = new int[blockCount];
        for (int i = 0; i < blockCount; i++) {
            int start = i * blockSize;
            int end = Math.min(start + blockSize, arr.length);
            int max = Integer.MIN_VALUE;
            for (int j = start; j < end; j++) max = Math.max(max, arr[j]);
            maxValues[i] = max;
        }
    }

    public int search(int target) {
        // 1. 在索引表中二分查找
        int blockIdx = 0;
        while (blockIdx < maxValues.length && maxValues[blockIdx] < target) {
            blockIdx++;
        }
        if (blockIdx >= maxValues.length) return -1;
        // 2. 在块内顺序查找
        int start = blockIdx * blockSize;
        int end = Math.min(start + blockSize, arr.length);
        for (int i = start; i < end; i++) {
            if (arr[i] == target) return i;
        }
        return -1;
    }
}
// 时间：O(sqrt(n))（块大小为 sqrt(n) 时最优）
```

```java
int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> window = new HashMap<>();
    int left = 0, maxLen = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);

String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>();
    Map<Character, Integer> window = new HashMap<>();
    for (char c : t.toCharArray()) need.put(c, need.getOrDefault(c, 0) + 1);

    int left = 0, valid = 0;
    int start = 0, minLen = Integer.MAX_VALUE;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (need.containsKey(c)) {
            window.put(c, window.getOrDefault(c, 0) + 1);
            if (window.get(c).equals(need.get(c))) valid++;
        }
        // 收缩窗口
        while (valid == need.size()) {
            if (right - left + 1 < minLen) {
                start = left;
                minLen = right - left + 1;
            }
            char d = s.charAt(left);
            if (need.containsKey(d)) {
                if (window.get(d).equals(need.get(d))) valid--;
                window.put(d, window.get(d) - 1);
            }
            left++;
        }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
}
```

```java
        result.add(new ArrayList<>(path));
        return;

        used[i] = true;

}
```

```java
    for (int i = 0; i < n; i++) {
        // 逆序遍历，保证每个物品只选一次
        for (int w = capacity; w >= weights[i]; w--) {
            dp[w] = Math.max(dp[w], dp[w - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
// 时间：O(n * capacity)，空间：O(capacity)
```

**多重背包**：

```java
// 每个物品有有限个数
int knapsackMultiple(int[] weights, int[] values, int[] counts, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weights.length; i++) {
        // 二进制拆分优化
        int num = counts[i];
        for (int k = 1; num > 0; k *= 2) {
            int amount = Math.min(k, num);
            num -= amount;
            int w = weights[i] * amount;
            int v = values[i] * amount;
            for (int j = capacity; j >= w; j--) {
                dp[j] = Math.max(dp[j], dp[j - w] + v);
            }
        }
    }
    return dp[capacity];
}
```

```java
    for (int i = 0; i <= m; i++) dp[i][0] = i; // 删除 i 次
    for (int j = 0; j <= n; j++) dp[0][j] = j; // 插入 j 次

                             dp[i][j - 1])  // 插入
                );
            }
        }
    }
    return dp[m][n];
}
```

```java
    List<List<Integer>> result = new ArrayList<>();
    Arrays.sort(nums);
    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i - 1]) continue; // 去重
        int left = i + 1, right = nums.length - 1;
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];

                result.add(Arrays.asList(nums[i], nums[left], nums[right]));

                left++; right--;
            } else if (sum < 0) left++;
            else right--;
        }
    }
    return result;
```

```java
int trap(int[] height) {
    int n = height.length;
    if (n == 0) return 0;
    int[] leftMax = new int[n], rightMax = new int[n];
    leftMax[0] = height[0];
    for (int i = 1; i < n; i++) leftMax[i] = Math.max(leftMax[i - 1], height[i]);
    rightMax[n - 1] = height[n - 1];
    for (int i = n - 2; i >= 0; i--) rightMax[i] = Math.max(rightMax[i + 1], height[i]);

    int water = 0;
    for (int i = 0; i < n; i++) {
        water += Math.min(leftMax[i], rightMax[i]) - height[i];
    }
    return water;
}

// 方法2：双指针（最优）
int trap2(int[] height) {
    int left = 0, right = height.length - 1;
    int leftMax = 0, rightMax = 0;
    int water = 0;
    while (left < right) {
        if (height[left] < height[right]) {
            if (height[left] >= leftMax) leftMax = height[left];
            else water += leftMax - height[left];
            left++;
        } else {
            if (height[right] >= rightMax) rightMax = height[right];
            else water += rightMax - height[right];
            right--;
        }
    }
    return water;
}
// 时间：O(n)，空间：O(1)
```

## 10.7 常见面试题汇总

### Q1: 数组和链表的区别？ArrayList 和 LinkedList 怎么选？

**A:** 数组在内存中**连续存储**，支持 O(1) 随机访问，但插入删除需要移动元素 O(n)。链表在内存中**离散存储**，插入删除 O(1)（已知位置），但随机访问 O(n)。实际开发中**优先选择 ArrayList**，原因：1) CPU 缓存对连续内存更友好；2) LinkedList 的每个节点需要额外存储指针，内存开销大；3) 即使在头部插入，ArrayList 的扩容复制在现代 JVM 中也很快。只有在频繁在头部/中间插入删除且不需要随机访问的场景才考虑 LinkedList。

### Q2: HashMap 的底层实现原理？

**A:** JDK 8 的 HashMap 由**数组 + 链表 + 红黑树**组成。通过 `hash(key) & (n-1)` 计算索引位置。发生哈希冲突时使用**拉链法**解决。当链表长度 >= 8 且数组长度 >= 64 时，链表转换为**红黑树**以提高查找效率。当红黑树节点数 <= 6 时退化回链表。默认初始容量 16，负载因子 0.75，扩容时容量翻倍。扩容时元素的位置要么不变，要么移动到 `原位置 + oldCap`。

### Q3: HashMap 为什么线程不安全？ConcurrentHashMap 是怎么保证线程安全的？

**A:** HashMap 在多线程环境下的问题：1) JDK 7 中并发扩容可能导致**环形链表**，引发死循环；2) 并发 put 可能导致**数据覆盖**。ConcurrentHashMap 在 JDK 8 中使用 **CAS + synchronized** 保证线程安全：只在当前桶的头节点上加锁，锁粒度细，并发性能好。初始化数组、扩容等操作也通过 CAS 保证原子性。

### Q4: HashMap 扩容机制是怎样的？

**A:** 当 `size > capacity * loadFactor` 时触发扩容。新容量为旧容量的 **2 倍**。扩容时重新计算每个元素的位置：`newIndex = hash & (newCapacity - 1)`。由于容量是 2 的幂次，元素的新位置只有两种可能：**原位置**或**原位置 + oldCap**（通过 `hash & oldCap` 判断）。链表在扩容时会拆分为两段，分别放到原位置和新位置，避免了像 JDK 7 那样重新计算哈希。

### Q5: 红黑树和 AVL 树的区别？

**A:** AVL 树是**严格平衡**的二叉搜索树，任意节点左右子树高度差不超过 1，查找效率高但插入删除旋转次数多。红黑树是**近似平衡**，最长路径不超过最短路径的 2 倍，插入删除最多 3 次旋转。HashMap 选择红黑树是因为：1) 正常情况下 HashMap 中链表不会太长，红黑树的近似平衡足够；2) 红黑树在插入删除时旋转少，性能更好；3) 红黑树的实现相对简单。

### Q6: 快速排序的原理？最坏情况怎么优化？

**A:** 快排使用**分治思想**：选择一个 pivot，将数组分为小于 pivot 和大于 pivot 两部分，递归排序。最坏情况 O(n^2) 发生在数组已有序且每次选到最大/最小值为 pivot。优化方法：1) **三数取中法**选 pivot，取左、中、右三个元素的中位数；2) **随机化**选 pivot；3) **三路快排**处理大量重复元素；4) 小数组切换**插入排序**（通常阈值 16）。

### Q7: 讲一下归并排序和快速排序的区别？

**A:** 归并排序是**自底向上**的分治，先递归拆分再合并，始终 O(n log n)，但需要 O(n) 额外空间，是**稳定**排序。快速排序是**自顶向下**的分治，先分区再递归，平均 O(n log n) 最坏 O(n^2)，原地排序 O(log n) 栈空间，是**不稳定**排序。Java 中 `Arrays.sort()` 对对象用 TimSort（归并变体），对基本类型用 DualPivotQuicksort。

### Q8: 二分查找的边界条件怎么处理？

**A:** 二分查找的关键是明确**搜索区间**和**循环条件**。`while (left <= right)` 对应闭区间 `[left, right]`，结束时 `left = right + 1`。`while (left < right)` 对应左闭右开区间 `[left, right)`，结束时 `left == right`。查找左边界时收缩右边界 `right = mid`，查找右边界时收缩左边界 `left = mid + 1`（注意向上取整 `mid = left + (right - left + 1) / 2` 防止死循环）。

### Q9: 什么是动态规划？什么样的题目适合用 DP？

**A:** 动态规划是将问题分解为**重叠子问题**，通过保存子问题的解避免重复计算。适用条件：1) **最优子结构**：问题的最优解包含子问题的最优解；2) **重叠子问题**：子问题会被重复计算；3) **无后效性**：当前状态只依赖之前的状态，不受后续决策影响。解题步骤：1) 定义状态（dp 数组的含义）；2) 推导状态转移方程；3) 确定初始条件和边界；4) 确定遍历顺序；5) 优化空间（滚动数组）。

### Q10: B+ 树为什么适合做数据库索引？

**A:** B+ 树相比 B 树和红黑树有以下优势：1) **树更矮**：B+ 树是多路搜索树，一个节点可以存多个索引，3-4 层就能存储百万级数据，减少磁盘 I/O；2) **非叶子节点不存数据**：每个节点能存更多索引，进一步降低树高；3) **叶子节点链表**：支持高效的范围查询，只需遍历链表；4) **查询性能稳定**：每次查询都到叶子节点，时间确定。InnoDB 中一个页 16KB，B+ 树高度为 3 时可存储约 2千万条记录。

### Q11: 如何判断链表有环？找到环的入口？

**A:** 使用**快慢指针**：快指针每次走两步，慢指针每次走一步。如果快指针追上慢指针则有环。找入口：相遇后，一个指针从头开始，另一个从相遇点开始，两者都以每次一步的速度前进，再次相遇的位置就是环的入口。数学证明：设头到入口距离为 a，入口到相遇点距离为 b，相遇点到入口距离为 c。慢指针走了 a+b，快指针走了 a+b+k(b+c)=a+(k+1)b+kc。由于快指针是慢指针的 2 倍：2(a+b)=a+(k+1)b+kc → a=c+(k-1)(b+c)。因此从头走到入口的距离等于从相遇点走 k-1 圈加 c 的距离。

### Q12: 堆排序的原理？应用场景？

**A:** 堆排序利用**完全二叉树**的性质：1) 建堆：从最后一个非叶子节点开始自上而下调整，时间 O(n)；2) 排序：每次将堆顶（最大值）与末尾交换，然后对堆顶做下沉调整，时间 O(n log n)。应用场景：1) **Top K 问题**：维护大小为 K 的堆；2) **优先队列**；3) **合并有序流**；4) **中位数查找**（维护一个大顶堆和一个小顶堆）。Java 中 `PriorityQueue` 默认是最小堆，通过自定义比较器可以实现最大堆。

### Q13: 什么是贪心算法？和动态规划的区别？

**A:** 贪心算法在每一步选择中都做出**当前最优**的选择，期望最终得到全局最优解。与动态规划的区别：贪心做出选择后**不回头**，而 DP 会**考虑所有可能**的子问题。贪心适用条件：问题具有**贪心选择性质**（局部最优能导致全局最优）和**最优子结构**。典型应用：区间调度、Huffman 编码、Dijkstra 最短路径、Kruskal/Prim 最小生成树。贪心不一定能得到最优解，需要严格证明。

### Q14: 什么是最小生成树？Kruskal 和 Prim 的区别？

**A:** 最小生成树是连通加权无向图中权值之和最小的生成树。Kruskal 算法：按边权排序，依次选择不形成环的最小边，使用**并查集**判断连通性，时间 O(E log E)，适合**稀疏图**。Prim 算法：从一个顶点开始，每次选择与当前树相连的最小权边扩展，使用**优先队列**优化，时间 O((V+E) log V)，适合**稠密图**。两者都是贪心算法，都能保证得到最优解。

### Q15: 讲一下 LRU 缓存的实现原理？

**A:** LRU（Least Recently Used）淘汰最近最少使用的数据。使用 **HashMap + 双向链表** 实现：HashMap 提供 O(1) 查找，双向链表维护访问顺序。get 操作：访问后将节点移动到链表头部。put 操作：新节点插入链表头部，超出容量时删除链表尾部节点。时间复杂度均为 O(1)。Java 中可以使用 `LinkedHashMap` 并设置 `accessOrder=true`，或者继承它重写 `removeEldestEntry` 方法。

```java
class LRUCache {
    private int capacity;
    private Map<Integer, Node> map;
    private Node head, tail; // 哨兵节点

    class Node {
        int key, value;
        Node prev, next;
    }

    public LRUCache(int capacity) {
        this.capacity = capacity;
        map = new HashMap<>();
        head = new Node(); tail = new Node();
        head.next = tail; tail.prev = head;
    }

    public int get(int key) {
        if (!map.containsKey(key)) return -1;
        Node node = map.get(key);
        removeNode(node);
        addToHead(node);
        return node.value;
    }

    public void put(int key, int value) {
        if (map.containsKey(key)) {
            Node node = map.get(key);
            node.value = value;
            removeNode(node);
            addToHead(node);
        } else {
            Node node = new Node();
            node.key = key; node.value = value;
            map.put(key, node);
            addToHead(node);
            if (map.size() > capacity) {
                Node removed = tail.prev;
                removeNode(removed);
                map.remove(removed.key);
            }
        }
    }

    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    private void addToHead(Node node) {
        node.next = head.next;
        node.prev = head;
        head.next.prev = node;
        head.next = node;
    }
}
```

### Q16: 如何实现一个线程安全的单例模式？

```java
public class Singleton {
    private static volatile Singleton instance;
    private Singleton() {}
    public static Singleton getInstance() {
        if (instance == null) {
            synchronized (Singleton.class) {
                if (instance == null) {

                }
            }
        }
        return instance;
    }
```

### Q17: 手写一个生产者-消费者模型？

**A:**

```java
class ProducerConsumer {
    private final int[] buffer;
    private int count = 0;
    private int capacity;
    private int putIdx = 0, takeIdx = 0;

    public ProducerConsumer(int capacity) {
        this.capacity = capacity;
        this.buffer = new int[capacity];
    }

    public synchronized void produce(int item) throws InterruptedException {
        while (count == capacity) { // 缓冲区满，等待
            wait();
        }
        buffer[putIdx] = item;
        putIdx = (putIdx + 1) % capacity;
        count++;
        notifyAll(); // 通知消费者
    }

    public synchronized int consume() throws InterruptedException {
        while (count == 0) { // 缓冲区空，等待
            wait();
        }
        int item = buffer[takeIdx];
        takeIdx = (takeIdx + 1) % capacity;
        count--;
        notifyAll(); // 通知生产者
        return item;
    }
}

// 使用 BlockingQueue 更简洁
class ProducerConsumer2 {
    private BlockingQueue<Integer> queue = new ArrayBlockingQueue<>(10);

    public void produce(int item) throws InterruptedException {
        queue.put(item); // 队列满时自动阻塞
    }

    public int consume() throws InterruptedException {
        return queue.take(); // 队列空时自动阻塞
    }
}
```

### Q18: 手写快速排序并分析时间复杂度？

**A:** 见 10.3.2 快速排序。时间复杂度分析：每次 partition 将数组分为两部分，理想情况下每次分成等长的两部分，递归树高度为 log n，每层处理 O(n)，总计 O(n log n)。最坏情况（已排序数组 + 固定选 pivot）递归树退化为链状，高度为 n，总计 O(n^2)。空间复杂度为递归栈深度 O(log n)（平均），最坏 O(n)。快速排序是不稳定排序，因为 partition 过程中元素的相对位置可能改变。

### Q19: 什么是 Top K 问题？有哪些解法？

**A:** Top K 问题是从大量数据中找出最大/最小的 K 个元素。解法：1) **排序**：O(n log n)，适合 n 不大的情况；2) **最小堆**：维护大小为 K 的最小堆，遍历数据，堆满后新元素大于堆顶则替换，时间 O(n log K)，空间 O(K)，适合**流式数据**；3) **快速选择**：基于快排的 partition，平均 O(n)，最坏 O(n^2)，空间 O(1)；4) **计数排序/桶排序**：如果数据范围有限，可以 O(n) 解决。实际工程中，数据量极大时通常用**最小堆**。

### Q20: 二叉树的前中后序遍历分别有什么应用？

**A:** **前序遍历**（根-左-右）：1) 序列化/反序列化二叉树；2) 复制二叉树；3) 表达式树的前缀表达式。**中序遍历**（左-根-右）：1) BST 中得到有序序列；2) 查找 BST 中第 K 小的元素。**后序遍历**（左-右-根）：1) 计算目录大小（先算子目录再算当前）；2) 释放二叉树内存；3) 表达式树的后缀表达式求值。**层序遍历**：1) 按层打印二叉树；2) 求二叉树的最大宽度；3) 最短路径问题（无权图 BFS）。
