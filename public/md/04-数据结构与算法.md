## 一、复杂度分析

### 1.1 时间复杂度

时间复杂度描述算法运行时间随输入规模增长的变化趋势，使用大 O 表示法。

#### 常见时间复杂度

| 复杂度 | 名称 | 典型场景 | n=100 耗时 |
|--------|------|----------|------------|
| O(1) | 常数 | 数组下标访问、哈希查找 | 1 |
| O(log n) | 对数 | 二分查找、平衡BST | 7 |
| O(n) | 线性 | 遍历数组/链表 | 100 |
| O(n log n) | 线性对数 | 归并排序、快速排序 | 700 |
| O(n^2) | 平方 | 冒泡排序、嵌套循环 | 10000 |
| O(n^3) | 立方 | 三层嵌套循环、矩阵乘法 | 1000000 |
| O(2^n) | 指数 | 子集枚举（无剪枝） | 1.27e30 |
| O(n!) | 阶乘 | 全排列（无剪枝） | 9.33e157 |

#### 时间复杂度计算规则

```javascript
// O(1) - 常数时间
function getFirst(arr) {
    return arr[0]; // 只执行一次操作
}

// O(n) - 线性时间
function sum(arr) {
    let total = 0;
    for (let i = 0; i < arr.length; i++) { // 循环 n 次
        total += arr[i];
    }
    return total;
}

// O(n^2) - 平方时间
function bubbleSort(arr) {
    for (let i = 0; i < arr.length; i++) {        // O(n)
        for (let j = 0; j < arr.length - i - 1; j++) { // O(n)
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

// O(log n) - 对数时间
function binarySearch(arr, target) {
    let left = 0, right = arr.length - 1;
    while (left <= right) { // 每次范围减半
        const mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

### 1.2 空间复杂度

空间复杂度描述算法额外使用的内存空间随输入规模增长的趋势。

```javascript
// O(1) - 原地操作，只使用常数额外空间
function reverseArray(arr) {
    let left = 0, right = arr.length - 1;
    while (left < right) {
        [arr[left], arr[right]] = [arr[right], arr[left]];
        left++;
        right--;
    }
    return arr;
}

// O(n) - 需要额外线性空间
function copyArray(arr) {
    const result = []; // 额外开辟 n 大小空间
    for (let i = 0; i < arr.length; i++) {
        result.push(arr[i]);
    }
    return result;
}

// O(n^2) - 二维数组
function createMatrix(n) {
    const matrix = []; // n x n 矩阵
    for (let i = 0; i < n; i++) {
        matrix[i] = new Array(n).fill(0);
    }
    return matrix;
}
```

### 1.3 递归复杂度分析

递归算法的复杂度分析需要考虑**递归深度**和**每层的计算量**。

#### 递归树分析法

```javascript
// 斐波那契数列 - 递归实现
// 时间复杂度: O(2^n)，空间复杂度: O(n)（递归栈深度）
function fib(n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}

// 递归树分析:
//                    fib(5)
//                  /        \
//            fib(4)          fib(3)
//           /      \        /      \
//       fib(3)   fib(2)  fib(2)  fib(1)
//       /    \    /    \
//   fib(2) fib(1) fib(1) fib(0)
// 节点数 ≈ 2^n，所以时间复杂度为 O(2^n)
```

#### 主定理 (Master Theorem)

对于递推关系 `T(n) = aT(n/b) + O(n^d)`：

| 条件 | 时间复杂度 | 示例 |
|------|-----------|------|
| d < log_b(a) | O(n^log_b(a)) | T(n)=3T(n/2)+O(1) → O(n^1.58) |
| d = log_b(a) | O(n^d * log n) | 归并排序: T(n)=2T(n/2)+O(n) → O(n log n) |
| d > log_b(a) | O(n^d) | — |

```javascript
// 带备忘录的斐波那契 - 优化后
// 时间复杂度: O(n)，空间复杂度: O(n)
function fibMemo(n, memo = {}) {
    if (n <= 1) return n;
    if (memo[n] !== undefined) return memo[n];
    memo[n] = fibMemo(n - 1, memo) + fibMemo(n - 2, memo);
    return memo[n];
}

// 迭代实现 - 进一步优化空间
// 时间复杂度: O(n)，空间复杂度: O(1)
function fibIterative(n) {
    if (n <= 1) return n;
    let prev = 0, curr = 1;
    for (let i = 2; i <= n; i++) {
        [prev, curr] = [curr, prev + curr];
    }
    return curr;
}
```

### 2.1 双指针

双指针是处理数组/字符串的高频技巧，分为**对撞指针**和**快慢指针**。

#### 模板

```javascript
// 对撞指针模板（适用于有序数组）
function twoPointersTemplate(arr) {
    let left = 0, right = arr.length - 1;
    while (left < right) {
        if (满足某个条件) {
            left++;
            // 或 right--
        } else if (满足另一个条件) {
            right--;
        } else {
            // 找到答案
            return [arr[left], arr[right]];
        }
    }
}
```

#### LeetCode 167. 两数之和 II - 输入有序数组 [简单]

**题目**: 给定一个已按升序排列的整数数组，找到两个数使它们之和等于目标值，返回它们的下标。

**思路**: 对撞指针，左指针从头开始，右指针从尾开始，根据和与目标值的关系移动指针。

```javascript
function twoSum(numbers, target) {
    let left = 0, right = numbers.length - 1;
    while (left < right) {
        const sum = numbers[left] + numbers[right];
        if (sum === target) {
            return [left + 1, right + 1]; // 题目要求下标从1开始
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return [-1, -1];
}
// 时间复杂度: O(n)，空间复杂度: O(1)
```

#### LeetCode 11. 盛最多水的容器 [中等]

**题目**: 给定 n 条垂直线，找出两条线使得它们与 x 轴构成的容器能容纳最多的水。

**思路**: 对撞指针，每次移动较短的那一边（因为移动较长边不可能增加面积）。

```javascript
function maxArea(height) {
    let left = 0, right = height.length - 1;
    let maxWater = 0;
    while (left < right) {
        const width = right - left;
        const h = Math.min(height[left], height[right]);
        maxWater = Math.max(maxWater, width * h);
        // 移动较短的一边
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    return maxWater;
}
// 时间复杂度: O(n)，空间复杂度: O(1)
```

#### LeetCode 15. 三数之和 [中等]

**题目**: 找出数组中所有和为 0 的不重复三元组。

**思路**: 排序 + 固定一个数 + 双指针找另外两个数，注意跳过重复元素。

```javascript
function threeSum(nums) {
    const result = [];
    nums.sort((a, b) => a - b);
    for (let i = 0; i < nums.length - 2; i++) {
        // 跳过重复元素
        if (i > 0 && nums[i] === nums[i - 1]) continue;
        // 剪枝：如果最小值已经大于0，不可能有解
        if (nums[i] > 0) break;
        let left = i + 1, right = nums.length - 1;
        while (left < right) {
            const sum = nums[i] + nums[left] + nums[right];
            if (sum === 0) {
                result.push([nums[i], nums[left], nums[right]]);
                // 跳过重复
                while (left < right && nums[left] === nums[left + 1]) left++;
                while (left < right && nums[right] === nums[right - 1]) right--;
                left++;
                right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return result;
}
// 时间复杂度: O(n^2)，空间复杂度: O(1)（忽略排序空间）
```

### 2.2 滑动窗口

滑动窗口用于处理连续子数组/子字符串问题，可以在 O(n) 时间内完成。

#### 模板

```javascript
// 滑动窗口通用模板
function slidingWindowTemplate(s) {
    const window = new Map(); // 或普通对象记录窗口内字符频率
    let left = 0, right = 0;
    let result = 0; // 根据题目定义结果

    while (right < s.length) {
        // 扩大窗口：将 s[right] 加入窗口
        const c = s[right];
        window.set(c, (window.get(c) || 0) + 1);
        right++;

        // 当窗口不满足条件时，收缩窗口
        while (窗口需要收缩的条件) {
            const d = s[left];
            window.set(d, window.get(d) - 1);
            if (window.get(d) === 0) window.delete(d);
            left++;
        }

        // 在这里更新结果（窗口满足条件时）
        result = Math.max(result, right - left); // 示例
    }
    return result;
}
```

#### LeetCode 3. 无重复字符的最长子串 [中等]

**题目**: 找出字符串中不含重复字符的最长子串的长度。

**思路**: 滑动窗口 + 哈希表记录字符最新位置，遇到重复字符时收缩左边界。

```javascript
function lengthOfLongestSubstring(s) {
    const charIndex = new Map(); // 字符 -> 最新索引
    let left = 0, maxLen = 0;

    for (let right = 0; right < s.length; right++) {
        const char = s[right];
        if (charIndex.has(char) && charIndex.get(char) >= left) {
            // 重复字符在窗口内，收缩左边界
            left = charIndex.get(char) + 1;
        }
        charIndex.set(char, right);
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
// 时间复杂度: O(n)，空间复杂度: O(min(m,n))，m为字符集大小
```

#### LeetCode 76. 最小覆盖子串 [困难]

**题目**: 给定字符串 s 和 t，找出 s 中包含 t 所有字符的最小子串。

**思路**: 滑动窗口，用哈希表记录需要的字符和窗口内已有的字符。

```javascript
function minWindow(s, t) {
    if (t.length > s.length) return "";

    const need = new Map();
    const window = new Map();
    for (const c of t) {
        need.set(c, (need.get(c) || 0) + 1);
    }

    let left = 0, right = 0;
    let valid = 0; // 窗口中满足 need 条件的字符数
    let start = 0, minLen = Infinity;

    while (right < s.length) {
        const c = s[right];
        right++;
        if (need.has(c)) {
            window.set(c, (window.get(c) || 0) + 1);
            if (window.get(c) === need.get(c)) {
                valid++;
            }
        }

        // 收缩窗口
        while (valid === need.size) {
            // 更新最小子串
            if (right - left < minLen) {
                start = left;
                minLen = right - left;
            }
            const d = s[left];
            left++;
            if (need.has(d)) {
                if (window.get(d) === need.get(d)) {
                    valid--;
                }
                window.set(d, window.get(d) - 1);
            }
        }
    }
    return minLen === Infinity ? "" : s.substring(start, start + minLen);
}
// 时间复杂度: O(|s| + |t|)，空间复杂度: O(|s| + |t|)
```

#### LeetCode 209. 长度最小的子数组 [中等]

```javascript
function minSubArrayLen(target, nums) {
    let left = 0, sum = 0, minLen = Infinity;
    for (let right = 0; right < nums.length; right++) {
        sum += nums[right];
        while (sum >= target) {
            minLen = Math.min(minLen, right - left + 1);
            sum -= nums[left];
            left++;
        }
```

### 2.3 前缀和

前缀和用于快速计算区间和，将 O(n) 的区间查询优化到 O(1)。

#### 模板

```javascript
// 一维前缀和
// prefix[i] = arr[0] + arr[1] + ... + arr[i-1]
// 区间 [l, r] 的和 = prefix[r+1] - prefix[l]

function buildPrefixSum(arr) {
    const prefix = new Array(arr.length + 1).fill(0);
    for (let i = 0; i < arr.length; i++) {
        prefix[i + 1] = prefix[i] + arr[i];
    }
    return prefix;
}

function rangeSum(prefix, left, right) {
    return prefix[right + 1] - prefix[left];
}
```

#### LeetCode 303. 区域和检索 - 数组不可变 [简单]

**题目**: 给定一个整数数组，求区间 [left, right] 内元素的和。

```javascript
class NumArray {
    constructor(nums) {
        this.prefix = new Array(nums.length + 1).fill(0);
        for (let i = 0; i < nums.length; i++) {
            this.prefix[i + 1] = this.prefix[i] + nums[i];
        }
    }

    sumRange(left, right) {
        return this.prefix[right + 1] - this.prefix[left];
    }
}
// 构造时间复杂度: O(n)，查询时间复杂度: O(1)，空间复杂度: O(n)
```

#### LeetCode 560. 和为 K 的子数组 [中等]

**题目**: 给定数组和整数 k，求连续子数组的和恰好等于 k 的个数。

**思路**: 前缀和 + 哈希表。若 prefix[j] - prefix[i] = k，则子数组 [i, j-1] 的和为 k。

```javascript
function subarraySum(nums, k) {
    const prefixCount = new Map();
    prefixCount.set(0, 1); // 前缀和为0出现1次
    let prefixSum = 0, count = 0;

    for (const num of nums) {
        prefixSum += num;
        // 查找是否存在 prefixSum - k
        if (prefixCount.has(prefixSum - k)) {
            count += prefixCount.get(prefixSum - k);
        }
        prefixCount.set(prefixSum, (prefixCount.get(prefixSum) || 0) + 1);
    }
    return count;
}
// 时间复杂度: O(n)，空间复杂度: O(n)
```

### 2.4 二分查找

二分查找在有序数据上以 O(log n) 的时间查找目标值。

#### 模板

```javascript
// 二分查找模板 - 寻找精确匹配
function binarySearch(nums, target) {
    let left = 0, right = nums.length - 1;
    while (left <= right) {
        const mid = left + Math.floor((right - left) / 2); // 防溢出
        if (nums[mid] === target) return mid;
        else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// 二分查找模板 - 寻找左边界（第一个 >= target 的位置）
function lowerBound(nums, target) {
    let left = 0, right = nums.length; // 注意 right 是 nums.length
    while (left < right) {
        const mid = left + Math.floor((right - left) / 2);
        if (nums[mid] < target) left = mid + 1;
        else right = mid;
    }
    return left;
}

// 二分查找模板 - 寻找右边界（最后一个 <= target 的位置）
function upperBound(nums, target) {
    let left = 0, right = nums.length;
    while (left < right) {
        const mid = left + Math.floor((right - left) / 2);
        if (nums[mid] <= target) left = mid + 1;
        else right = mid;
    }
    return left - 1;
}
```

#### LeetCode 33. 搜索旋转排序数组 [中等]

**题目**: 在旋转后的有序数组中搜索目标值。

**思路**: 判断哪一半是有序的，再判断目标值是否在有序的那一半中。

```javascript
function search(nums, target) {
    let left = 0, right = nums.length - 1;
    while (left <= right) {
        const mid = left + Math.floor((right - left) / 2);
        if (nums[mid] === target) return mid;

        // 判断左半部分是否有序
        if (nums[left] <= nums[mid]) {
            if (nums[left] <= target && target < nums[mid]) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        } else {
            // 右半部分有序
            if (nums[mid] < target && target <= nums[right]) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
    }
    return -1;
}
// 时间复杂度: O(log n)，空间复杂度: O(1)
```

#### LeetCode 4. 寻找两个正序数组的中位数 [困难]

**题目**: 给定两个大小分别为 m 和 n 的正序数组，找出它们的中位数。

**思路**: 二分查找较短数组，找到合适的分割点使得左半部分最大值 <= 右半部分最小值。

```javascript
function findMedianSortedArrays(nums1, nums2) {
    // 确保 nums1 是较短的数组
    if (nums1.length > nums2.length) {
        [nums1, nums2] = [nums2, nums1];
    }
    const m = nums1.length, n = nums2.length;
    let left = 0, right = m;

    while (left <= right) {
        const i = left + Math.floor((right - left) / 2); // nums1 的分割点
        const j = Math.floor((m + n + 1) / 2) - i;       // nums2 的分割点

        const nums1LeftMax = i === 0 ? -Infinity : nums1[i - 1];
        const nums1RightMin = i === m ? Infinity : nums1[i];
        const nums2LeftMax = j === 0 ? -Infinity : nums2[j - 1];
        const nums2RightMin = j === n ? Infinity : nums2[j];

        if (nums1LeftMax <= nums2RightMin && nums2LeftMax <= nums1RightMin) {
            // 找到正确的分割点
            if ((m + n) % 2 === 1) {
                return Math.max(nums1LeftMax, nums2LeftMax);
            }
            return (Math.max(nums1LeftMax, nums2LeftMax) + Math.min(nums1RightMin, nums2RightMin)) / 2;
        } else if (nums1LeftMax > nums2RightMin) {
            right = i - 1;
        } else {
            left = i + 1;
        }
    }
    throw new Error("输入数组不是有效的有序数组");
}
// 时间复杂度: O(log(min(m,n)))，空间复杂度: O(1)
```

## 三、链表

### 3.1 链表基础

```javascript
// 单链表节点定义
class ListNode {
    constructor(val, next = null) {
        this.val = val;
        this.next = next;
    }
}

// 双链表节点定义
class DoublyListNode {
    constructor(val, prev = null, next = null) {
        this.val = val;
        this.prev = prev;
        this.next = next;
```

### 3.3 快慢指针

#### LeetCode 141. 环形链表 [简单]

**题目**: 判断链表中是否有环。

**思路**: 快慢指针，快指针每次走两步，慢指针每次走一步。如果有环，快指针最终会追上慢指针。

```javascript
function hasCycle(head) {
    let slow = head, fast = head;
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) return true;
    }
    return false;
}
// 时间复杂度: O(n)，空间复杂度: O(1)
```

#### LeetCode 142. 环形链表 II [中等]

**题目**: 找到链表中环的入口节点。

**思路**: 快慢指针相遇后，将一个指针移回起点，两个指针以相同速度前进，再次相遇即为入口。

```javascript
function detectCycle(head) {
    let slow = head, fast = head;
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) {
            // 找到相遇点，找入口
            let ptr1 = head, ptr2 = slow;
            while (ptr1 !== ptr2) {
                ptr1 = ptr1.next;
                ptr2 = ptr2.next;
            }
            return ptr1;
        }
    }
    return null;
}
// 时间复杂度: O(n)，空间复杂度: O(1)
// 证明：设起点到入口距离为 a，入口到相遇点距离为 b，环长为 c
// 快指针走了 a + b + kc，慢指针走了 a + b
// 2(a + b) = a + b + kc → a = kc - b = (k-1)c + (c-b)
// 即从起点走 a 步等于从相遇点走 c-b 步（再走 b 步到入口，但方向相反所以是 c-b）
```

#### LeetCode 23. 合并 K 个升序链表 [困难]

**思路**: 使用优先队列（最小堆），每次取出最小的节点。

```javascript
class MinHeap {
    constructor() { this.heap = []; }
    push(val) {
        this.heap.push(val);
        this._bubbleUp(this.heap.length - 1);
    }
    pop() {
        const top = this.heap[0];
        const last = this.heap.pop();
        if (this.heap.length > 0) {
            this.heap[0] = last;
            this._sinkDown(0);
        }
        return top;
    }
    get size() { return this.heap.length; }
    _bubbleUp(i) {
        while (i > 0) {
            const parent = Math.floor((i - 1) / 2);
            if (this.heap[parent].val <= this.heap[i].val) break;
            [this.heap[parent], this.heap[i]] = [this.heap[i], this.heap[parent]];
            i = parent;
        }
    }
    _sinkDown(i) {
        const n = this.heap.length;
        while (true) {
            let smallest = i;
            const left = 2 * i + 1, right = 2 * i + 2;
            if (left < n && this.heap[left].val < this.heap[smallest].val) smallest = left;
            if (right < n && this.heap[right].val < this.heap[smallest].val) smallest = right;
            if (smallest === i) break;
            [this.heap[smallest], this.heap[i]] = [this.heap[i], this.heap[smallest]];
            i = smallest;
        }
    }
}

        const node = minHeap.pop();

        current = current.next;
```

```javascript
function trap(height) {
    let water = 0;
    const stack = []; // 单调递减栈，存储索引

    for (let i = 0; i < height.length; i++) {
        while (stack.length > 0 && height[i] > height[stack[stack.length - 1]]) {
            const bottom = stack.pop();
            if (stack.length === 0) break;
            const left = stack[stack.length - 1];
            const width = i - left - 1;
            const h = Math.min(height[left], height[i]) - height[bottom];
            water += width * h;
        }
        stack.push(i);
    }
    return water;
}
```

```javascript
    const n = heights.length;
    const left = new Array(n).fill(-1);  // 左边界
    const right = new Array(n).fill(n);  // 右边界
    const stack = [];

            stack.pop();
        }
        left[i] = stack.length > 0 ? stack[stack.length - 1] : -1;
        stack.push(i);
    }

    stack.length = 0;
    // 计算每个柱子的右边界（第一个小于它的位置）
    for (let i = n - 1; i >= 0; i--) {

            stack.pop();

        stack.push(i);
```

```javascript
    const pivotIndex = partition(nums, left, right);
    if (pivotIndex === k) return nums[k];
    return pivotIndex < k ? quickSelect(nums, pivotIndex + 1, right, k)

            [nums[i], nums[j]] = [nums[j], nums[i]];
            i++;
        }
```

#### LeetCode 200. 岛屿数量 [中等]

```javascript
    if (grid.length === 0) return 0;
    const rows = grid.length, cols = grid[0].length;
    let count = 0;

    function bfs(r, c) {
        const queue = [[r, c]];
        grid[r][c] = '0'; // 标记为已访问
        const directions = [[-1,0],[1,0],[0,-1],[0,1]];

        while (queue.length > 0) {
            const [cr, cc] = queue.shift();
            for (const [dr, dc] of directions) {
                const nr = cr + dr, nc = cc + dc;
                if (nr >= 0 && nr < rows && nc >= 0 && nc < cols && grid[nr][nc] === '1') {
                    grid[nr][nc] = '0';
                    queue.push([nr, nc]);
                }
            }
        }
    }

    for (let r = 0; r < rows; r++) {
        for (let c = 0; c < cols; c++) {
            if (grid[r][c] === '1') {
                bfs(r, c);
                count++;
```

### 5.4 LeetCode 128. 最长连续序列 [中等]

**题目**: 给定未排序数组，找出最长连续元素序列的长度。要求 O(n) 时间复杂度。

**思路**: 将所有元素存入 Set，对每个序列的起始元素（即前一个元素不在 Set 中的元素）向后查找。

```javascript
function longestConsecutive(nums) {
    const numSet = new Set(nums);
    let longest = 0;

    for (const num of numSet) {
        // 只有当 num 是序列的起点时才向后查找
        if (!numSet.has(num - 1)) {
            let currentNum = num;
            let currentStreak = 1;

            while (numSet.has(currentNum + 1)) {
                currentNum++;
                currentStreak++;
            }
            longest = Math.max(longest, currentStreak);
        }
    }
    return longest;
}
// 时间复杂度: O(n)，空间复杂度: O(n)
```

### 5.5 LeetCode 146. LRU 缓存 [中等]

**题目**: 设计 LRU（最近最少使用）缓存机制。

**思路**: 哈希表 + 双向链表。哈希表提供 O(1) 查找，双向链表维护访问顺序。

```javascript
class DLinkedNode {
    constructor(key = 0, val = 0) {
        this.key = key;
        this.val = val;
        this.prev = null;
        this.next = null;
    }
}

class LRUCache {
    constructor(capacity) {
        this.capacity = capacity;
        this.cache = new Map();
        // 使用哨兵节点简化边界处理
        this.head = new DLinkedNode();
        this.tail = new DLinkedNode();
        this.head.next = this.tail;
        this.tail.prev = this.head;
        this.size = 0;
    }

    get(key) {
        if (!this.cache.has(key)) return -1;
        const node = this.cache.get(key);
        this._moveToHead(node);
        return node.val;
    }

    put(key, value) {
        if (this.cache.has(key)) {
            const node = this.cache.get(key);
            node.val = value;
            this._moveToHead(node);
        } else {
            const newNode = new DLinkedNode(key, value);
            this.cache.set(key, newNode);
            this._addToHead(newNode);
            this.size++;
            if (this.size > this.capacity) {
                const removed = this._removeTail();
                this.cache.delete(removed.key);
                this.size--;
            }
        }
    }

    _addToHead(node) {
        node.prev = this.head;
        node.next = this.head.next;
        this.head.next.prev = node;
        this.head.next = node;
    }

    _removeNode(node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    _moveToHead(node) {
        this._removeNode(node);
        this._addToHead(node);
    }

    _removeTail() {
        const node = this.tail.prev;
        this._removeNode(node);
        return node;
    }
}
// 时间复杂度: get O(1), put O(1)；空间复杂度: O(capacity)
```

```javascript
        if (!node) return;

        traverse(node.left);

}
```

### 6.4 最近公共祖先 (LCA)

#### LeetCode 236. 二叉树的最近公共祖先 [中等]

```javascript
function lowestCommonAncestor(root, p, q) {
    if (!root || root === p || root === q) return root;
    const left = lowestCommonAncestor(root.left, p, q);
    const right = lowestCommonAncestor(root.right, p, q);

    if (left && right) return root; // p 和 q 分别在左右子树
    return left || right;           // p 或 q 在其中一棵子树中
}
```

### 7.4 拓扑排序

#### LeetCode 207. 课程表 [中等]

**题目**: 判断是否可以完成所有课程（课程之间有先修关系）。

**思路**: 拓扑排序（Kahn 算法），使用入度表 + BFS。

```javascript
function canFinish(numCourses, prerequisites) {
    // 构建图和入度表
    const graph = new Array(numCourses).fill(null).map(() => []);
    const inDegree = new Array(numCourses).fill(0);

    for (const [course, prereq] of prerequisites) {
        graph[prereq].push(course);
        inDegree[course]++;
    }

    // BFS 拓扑排序
    const queue = [];
    for (let i = 0; i < numCourses; i++) {
        if (inDegree[i] === 0) queue.push(i);
    }

    let completed = 0;
    while (queue.length > 0) {
        const course = queue.shift();
        completed++;
        for (const next of graph[course]) {
            inDegree[next]--;
            if (inDegree[next] === 0) {
                queue.push(next);
            }
        }
    }
    return completed === numCourses;
}
// 时间复杂度: O(V + E)，空间复杂度: O(V + E)
```

### 7.5 并查集 (Union-Find)

```javascript
class UnionFind {
    constructor(n) {
        this.parent = new Array(n).fill(null).map((_, i) => i);
        this.rank = new Array(n).fill(0);
        this.count = n; // 连通分量数
    }

    find(x) {
        // 路径压缩
        if (this.parent[x] !== x) {
            this.parent[x] = this.find(this.parent[x]);
        }
        return this.parent[x];
    }

    union(x, y) {
        const rootX = this.find(x);
        const rootY = this.find(y);
        if (rootX === rootY) return false;

        // 按秩合并
        if (this.rank[rootX] < this.rank[rootY]) {
            this.parent[rootX] = rootY;
        } else if (this.rank[rootX] > this.rank[rootY]) {
            this.parent[rootY] = rootX;
        } else {
            this.parent[rootY] = rootX;
            this.rank[rootX]++;
        }
        this.count--;
        return true;
    }

    connected(x, y) {
        return this.find(x) === this.find(y);
    }
}
```

```javascript
    dist[start] = 0;

    // 简单实现（使用数组模拟优先队列）
    for (let i = 0; i < n; i++) {
        // 找到未访问的最近节点
        let u = -1;
        for (let j = 0; j < n; j++) {
            if (!visited[j] && (u === -1 || dist[j] < dist[u])) {
                u = j;
            }
        }
        if (dist[u] === Infinity) break;
        visited[u] = true;

        // 更新邻居距离
        for (const { node: v, weight } of graph[u]) {
            if (dist[u] + weight < dist[v]) {
                dist[v] = dist[u] + weight;
            }
        }
    }
    return dist;
}

// 使用最小堆的优化版本
function dijkstraOptimized(graph, start) {
    const n = graph.length;
    const dist = new Array(n).fill(Infinity);
    dist[start] = 0;
    const minHeap = new MinHeap(); // 需要支持 {node, dist} 类型
    minHeap.push({ node: start, dist: 0 });

    while (minHeap.size > 0) {
        const { node: u, dist: d } = minHeap.pop();
        if (d > dist[u]) continue; // 跳过过时的记录

        for (const { node: v, weight } of graph[u]) {
            if (dist[u] + weight < dist[v]) {
                dist[v] = dist[u] + weight;
                minHeap.push({ node: v, dist: dist[v] });
            }
        }
    }
    return dist;
}
// 时间复杂度: O((V + E) log V)（堆优化），空间复杂度: O(V + E)
```

### 8.5 N 皇后

#### LeetCode 51. N 皇后 [困难]

**题目**: 在 n x n 棋盘上放置 n 个皇后，使其不能互相攻击。

```javascript
function solveNQueens(n) {
    const result = [];

    function backtrack(row, cols, diag1, diag2, board) {
        if (row === n) {
            result.push(board.map(r => r.join('')));
            return;
        }

        for (let col = 0; col < n; col++) {
            const d1 = row - col;      // 主对角线
            const d2 = row + col;      // 副对角线
            if (cols.has(col) || diag1.has(d1) || diag2.has(d2)) continue;

            // 放置皇后
            cols.add(col);
            diag1.add(d1);
            diag2.add(d2);
            board[row][col] = 'Q';

            backtrack(row + 1, cols, diag1, diag2, board);

            // 撤销
            cols.delete(col);
            diag1.delete(d1);
            diag2.delete(d2);
            board[row][col] = '.';
        }
    }

    const board = new Array(n).fill(null).map(() => new Array(n).fill('.'));
    backtrack(0, new Set(), new Set(), new Set(), board);
    return result;
}
// 时间复杂度: O(n!)，空间复杂度: O(n)
```

### 8.6 单词搜索

#### LeetCode 79. 单词搜索 [中等]

**题目**: 给定二维网格和单词，判断单词是否存在于网格中。

```javascript
function exist(board, word) {
    const rows = board.length, cols = board[0].length;

    function dfs(r, c, index) {
        if (index === word.length) return true;
        if (r < 0 || r >= rows || c < 0 || c >= cols || board[r][c] !== word[index]) {
            return false;
        }

        // 标记已访问
        const temp = board[r][c];
        board[r][c] = '#';

        const found = dfs(r + 1, c, index + 1) ||
                      dfs(r - 1, c, index + 1) ||
                      dfs(r, c + 1, index + 1) ||
                      dfs(r, c - 1, index + 1);

        board[r][c] = temp; // 恢复
        return found;
    }

    for (let r = 0; r < rows; r++) {
        for (let c = 0; c < cols; c++) {
            if (dfs(r, c, 0)) return true;
        }
    }
    return false;
}
// 时间复杂度: O(m * n * 4^L)，L 为单词长度；空间复杂度: O(L)
```

## 九、动态规划

### 9.1 动态规划核心思想

动态规划（DP）的本质是将问题分解为**重叠子问题**，通过存储子问题的解避免重复计算。

#### DP 解题步骤

```
1. 定义状态：dp[i] 或 dp[i][j] 代表什么含义？
2. 状态转移方程：dp[i] 如何从 dp[i-1]、dp[i-2] 等推导？
3. 初始化：dp[0]、dp[1] 等边界值是什么？
4. 遍历顺序：从前往后？从后往前？
5. 返回值：返回 dp[n] 还是 dp[n-1]？
```

#### 状态转移方程推导方法

```javascript
/**
 * 方法一：从暴力递归到记忆化搜索到递推
 *
 * 步骤1：写出暴力递归（穷举所有可能）
 * 步骤2：发现重叠子问题，加上 memo（记忆化搜索）
 * 步骤3：将递归改为自底向上的递推（动态规划）
 *
 * 方法二：直接定义状态和转移方程
 *
 * 步骤1：明确「状态」—— 原问题和子问题中变化的变量
 * 步骤2：明确「选择」—— 导致状态变化的行为
 * 步骤3：定义 dp 数组的含义
 * 步骤4：根据「选择」推导状态转移方程
 */
```

#### LeetCode 300. 最长递增子序列 (LIS) [中等]

**题目**: 找到数组中最长严格递增子序列的长度。

```javascript
// 方法1：动态规划 O(n^2)
function lengthOfLIS(nums) {
    const n = nums.length;
    const dp = new Array(n).fill(1); // dp[i] = 以 nums[i] 结尾的 LIS 长度
    let maxLen = 1;

    for (let i = 1; i < n; i++) {
        for (let j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
        maxLen = Math.max(maxLen, dp[i]);
    }
    return maxLen;
}
// 状态转移方程: dp[i] = max(dp[j] + 1) for all j < i where nums[j] < nums[i]
// 时间复杂度: O(n^2)，空间复杂度: O(n)

// 方法2：贪心 + 二分查找 O(n log n)
function lengthOfLISOptimized(nums) {
    const tails = []; // tails[i] = 长度为 i+1 的递增子序列的最小末尾元素
    for (const num of nums) {
        let left = 0, right = tails.length;
        while (left < right) {
            const mid = left + Math.floor((right - left) / 2);
            if (tails[mid] < num) left = mid + 1;
            else right = mid;
        }
        if (left === tails.length) tails.push(num);
        else tails[left] = num;
    }
    return tails.length;
}
// 时间复杂度: O(n log n)，空间复杂度: O(n)
```

### 9.5 背包问题

#### 0-1 背包问题

**题目**: 有 n 个物品，每个物品有重量和价值，背包容量为 W，求能装入的最大价值。

```javascript
function knapsack01(weights, values, W) {
    const n = weights.length;
    // dp[j] = 容量为 j 时的最大价值
    const dp = new Array(W + 1).fill(0);

    for (let i = 0; i < n; i++) {

    return dp[W];
}
// 状态转移方程: dp[j] = max(dp[j], dp[j - weights[i]] + values[i])
```

```javascript
    // dp[i][j] = 到达 (i,j) 的路径数
    const dp = new Array(m).fill(null).map(() => new Array(n).fill(1));

    for (let i = 1; i < m; i++) {
        for (let j = 1; j < n; j++) {
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
        }
    }
    return dp[m - 1][n - 1];
}
// 状态转移方程: dp[i][j] = dp[i-1][j] + dp[i][j-1]
```

## 十、排序算法

### 10.1 快速排序

```javascript
function quickSort(arr, left = 0, right = arr.length - 1) {
    if (left >= right) return;

    // 三数取中法选择 pivot（优化）
    const mid = left + Math.floor((right - left) / 2);
    if (arr[left] > arr[mid]) [arr[left], arr[mid]] = [arr[mid], arr[left]];
    if (arr[left] > arr[right]) [arr[left], arr[right]] = [arr[right], arr[left]];
    if (arr[mid] > arr[right]) [arr[mid], arr[right]] = [arr[right], arr[mid]];

    const pivot = arr[mid];
    [arr[mid], arr[right - 1]] = [arr[right - 1], arr[mid]]; // pivot 放到 right-1

    let i = left, j = right - 1;
    while (true) {
        while (arr[++i] < pivot) {}
        while (arr[--j] > pivot) {}
        if (i >= j) break;
        [arr[i], arr[j]] = [arr[j], arr[i]];
    }
    [arr[i], arr[right - 1]] = [arr[right - 1], arr[i]]; // pivot 归位

    quickSort(arr, left, i - 1);
    quickSort(arr, i + 1, right);
}

// 简洁版本
function quickSortSimple(arr) {
    if (arr.length <= 1) return arr;
    const pivot = arr[Math.floor(arr.length / 2)];
    const left = arr.filter(x => x < pivot);
    const middle = arr.filter(x => x === pivot);
    const right = arr.filter(x => x > pivot);
    return [...quickSortSimple(left), ...middle, ...quickSortSimple(right)];
}
// 时间复杂度: 平均 O(n log n)，最坏 O(n^2)；空间复杂度: 平均 O(log n)
```

```javascript
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            result.push(left[i++]);

            result.push(right[j++]);
        }
```

### 10.3 堆排序

```javascript
function heapSort(arr) {
    const n = arr.length;

    // 建堆（从最后一个非叶子节点开始）
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }

    // 逐个提取最大元素
    for (let i = n - 1; i > 0; i--) {
        [arr[0], arr[i]] = [arr[i], arr[0]]; // 将最大值移到末尾
        heapify(arr, i, 0); // 重新调整堆
    }
    return arr;
}

function heapify(arr, n, i) {
    let largest = i;
    const left = 2 * i + 1;
    const right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;

    if (largest !== i) {
        [arr[i], arr[largest]] = [arr[largest], arr[i]];
        heapify(arr, n, largest);
    }
}
// 时间复杂度: O(n log n)，空间复杂度: O(1)
```

### 10.4 计数排序

```javascript
function countingSort(arr) {
    if (arr.length === 0) return arr;
    const max = Math.max(...arr);
    const min = Math.min(...arr);
    const range = max - min + 1;

    const count = new Array(range).fill(0);
    const output = new Array(arr.length);

    // 统计频率
    for (const num of arr) {
        count[num - min]++;
    }

    // 计算前缀和（确定每个元素的最终位置）
    for (let i = 1; i < range; i++) {
        count[i] += count[i - 1];
    }

    // 从后往前放置元素（保持稳定性）
    for (let i = arr.length - 1; i >= 0; i--) {
        output[count[arr[i] - min] - 1] = arr[i];
        count[arr[i] - min]--;
    }
    return output;
}
// 时间复杂度: O(n + k)，k 为数值范围；空间复杂度: O(n + k)
```

```javascript
    let start = 0, maxLen = 1;

    function expandAroundCenter(left, right) {
        while (left >= 0 && right < s.length && s[left] === s[right]) {
            left--;
            right++;
        }
        return [left + 1, right - left - 1]; // [起始位置, 长度]
    }

    for (let i = 0; i < s.length; i++) {
        const [l1, len1] = expandAroundCenter(i, i);     // 奇数长度
        const [l2, len2] = expandAroundCenter(i, i + 1); // 偶数长度
        if (len1 > maxLen) { start = l1; maxLen = len1; }
        if (len2 > maxLen) { start = l2; maxLen = len2; }

}
```

```javascript
            window.set(c, (window.get(c) || 0) + 1);
            if (window.get(c) === need.get(c)) valid++;
        }
        while (valid === need.size) {
            if (right - left < minLen) { start = left; minLen = right - left; }
            const d = s[left++];
            if (need.has(d)) {
                if (window.get(d) === need.get(d)) valid--;
                window.set(d, window.get(d) - 1);
            }
        }
    }
    return minLen === Infinity ? "" : s.substring(start, start + minLen);
}
// 时间复杂度: O(|s| + |t|)，空间复杂度: O(|s| + |t|)
```

### 20. LeetCode 215. 数组中的第 K 个最大元素 [中等]

**思路**: 快速选择算法。

```javascript
function findKthLargest(nums, k) {
    const target = nums.length - k;
    function quickSelect(left, right) {
        const pivot = nums[right];
        let i = left;
        for (let j = left; j < right; j++) {
            if (nums[j] <= pivot) { [nums[i], nums[j]] = [nums[j], nums[i]]; i++; }
        }
        [nums[i], nums[right]] = [nums[right], nums[i]];
        if (i === target) return nums[i];
        return i < target ? quickSelect(i + 1, right) : quickSelect(left, i - 1);
    }
    return quickSelect(0, nums.length - 1);
}
// 时间复杂度: 平均 O(n)，最坏 O(n^2)；空间复杂度: O(1)
```

### 22. LeetCode 300. 最长递增子序列 [中等]

```javascript
function lengthOfLIS(nums) {
    const tails = [];
    for (const num of nums) {
        let left = 0, right = tails.length;
        while (left < right) {
            const mid = left + Math.floor((right - left) / 2);
            if (tails[mid] < num) left = mid + 1;
            else right = mid;
        }
        if (left === tails.length) tails.push(num);
        else tails[left] = num;
    }
    return tails.length;
}
```

### 24. LeetCode 337. 打家劫舍 III [中等]

**题目**: 在二叉树中不能偷相邻节点，求最大金额。

```javascript
function rob(root) {
    function dfs(node) {
        if (!node) return [0, 0]; // [不偷当前节点, 偷当前节点]
        const left = dfs(node.left);
        const right = dfs(node.right);
        return [
            Math.max(left[0], left[1]) + Math.max(right[0], right[1]), // 不偷
            node.val + left[0] + right[0] // 偷
        ];
    }
    const result = dfs(root);
    return Math.max(result[0], result[1]);
}
// 时间复杂度: O(n)，空间复杂度: O(h)
```

### 25. LeetCode 394. 字符串解码 [中等]

**题目**: 给定编码字符串，返回解码后的字符串。如 "3[a2[c]]" → "accaccacc"。

```javascript
function decodeString(s) {
    const stack = [];
    let currentNum = 0, currentStr = '';

    for (const char of s) {
        if (char >= '0' && char <= '9') {
            currentNum = currentNum * 10 + parseInt(char);
        } else if (char === '[') {
            stack.push(currentStr);
            stack.push(currentNum);
            currentStr = '';
            currentNum = 0;
        } else if (char === ']') {
            const num = stack.pop();
            const prevStr = stack.pop();
            currentStr = prevStr + currentStr.repeat(num);
        } else {
            currentStr += char;
        }
    }
    return currentStr;
}
// 时间复杂度: O(n)，空间复杂度: O(n)
```

### 26. LeetCode 416. 分割等和子集 [中等]

**思路**: 0-1 背包问题，判断是否存在子集和等于总和的一半。

```javascript
function canPartition(nums) {
    const sum = nums.reduce((a, b) => a + b, 0);
    if (sum % 2 !== 0) return false;
    const target = sum / 2;
    const dp = new Array(target + 1).fill(false);
    dp[0] = true;

    for (const num of nums) {
        for (let j = target; j >= num; j--) {
            dp[j] = dp[j] || dp[j - num];
        }
    }
    return dp[target];
}
// 时间复杂度: O(n * target)，空间复杂度: O(target)
```

### 29. LeetCode 72. 编辑距离 [中等]

```javascript
function minDistance(word1, word2) {
    const m = word1.length, n = word2.length;
    const dp = new Array(m + 1).fill(null).map(() => new Array(n + 1).fill(0));
    for (let i = 0; i <= m; i++) dp[i][0] = i;
    for (let j = 0; j <= n; j++) dp[0][j] = j;
    for (let i = 1; i <= m; i++) {
        for (let j = 1; j <= n; j++) {
            if (word1[i-1] === word2[j-1]) dp[i][j] = dp[i-1][j-1];
            else dp[i][j] = 1 + Math.min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]);
        }
    }
    return dp[m][n];
}
// 时间复杂度: O(m * n)，空间复杂度: O(m * n)
```

### 30. LeetCode 239. 滑动窗口最大值 [困难]

```javascript
function maxSlidingWindow(nums, k) {
    const result = [];
    const deque = [];
    for (let i = 0; i < nums.length; i++) {
        while (deque.length > 0 && deque[0] <= i - k) deque.shift();
        while (deque.length > 0 && nums[deque[deque.length - 1]] < nums[i]) deque.pop();
        deque.push(i);
        if (i >= k - 1) result.push(nums[deque[0]]);
    }
    return result;
}
```
