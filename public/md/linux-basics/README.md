# Linux 基础知识

Linux 面试和实战学习项目，包含命令、系统管理和 Shell 脚本。

## 文件结构

```
├── 01-commands.md    # 基础命令
├── 02-system.md     # 系统管理
├── 03-shell.md      # Shell 脚本
├── 04-interview.md  # 面试问答
└── README.md
```

## 内容概览

### 01-commands.md - 基础命令
- 文件操作 (ls, cat, find, chmod)
- 文本处理 (grep, awk, sed)
- 网络操作 (curl, ping, netstat)
- 压缩解压 (tar, zip)

### 02-system.md - 系统管理
- 用户和权限管理
- 服务管理 (systemctl)
- 定时任务 (crontab)
- 防火墙 (ufw, iptables)
- 性能监控 (top, vmstat)

### 03-shell.md - Shell 脚本
- 变量和数组
- 条件判断和循环
- 函数定义
- 字符串操作

### 04-interview.md - 面试问答
- Linux 启动过程
- 文件系统结构
- 硬链接 vs 软链接
- TCP 三次握手
- 性能排查
- 常见面试题

## 快速开始

```bash
# 在终端中运行命令练习
cat 01-commands.md

# 学习 Shell 脚本
cat 03-shell.md
```

## 核心命令速记

```bash
# 文件
ls -la, cat, head, tail, grep, find

# 进程
ps, top, kill, pgrep

# 网络
curl, ping, netstat, lsof

# 系统
df, du, free, uptime

# 权限
chmod, chown
```
