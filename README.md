# AIShow

一个基于 **Next.js + React + TypeScript** 的大模型学习可视化网站。

## 功能

- 按 **基础 / 进阶 / 实战** 三层展示关键知识概念
- 每个知识点包含：形象类比、核心说明、学习要点
- 提供从目标到复盘的 **操作路径时间线**
- 深色沉浸式 UI，适合长期记录与复盘

## 技术栈

- Next.js 15 (App Router)
- React 19
- TypeScript
- Framer Motion（动画）
- Lucide React（图标）

## 启动

```bash
npm install
npm run dev
```

打开 `http://localhost:3000` 查看。
